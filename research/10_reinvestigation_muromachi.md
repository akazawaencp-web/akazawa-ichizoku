## 重要な新発見（総括）

### 1. 「四宮の頭」の意味が判明
秀國の「四宮の頭」は、武水別神社（信濃国四宮）の神領である**四宮荘**の管理者・頭役を意味する。四宮荘は更級郡に所在し、赤沢氏が1335年の中先代の乱以来、地頭職を務めていた土地である。

### 2. 大塔合戦（1400年）に赤沢秀国が参戦
応永7年（1400年）の大塔合戦において、赤沢秀国が塩崎城主として小笠原長秀方で参陣していたことが複数の史料で確認された。「赤沢秀國・対馬守」が系図上の同一人物である可能性が極めて高い。

### 3. 武田分限帳に赤沢氏が記録
武田分限帳（永禄12年〜元亀4年頃）に「赤沢」が**信州先方衆**として40騎の軍役で記録されている。館は**松本市浅間温泉**に所在。これは信基・清純の「子孫は信州にあり」との記録に合致する可能性がある。

### 4. 赤沢氏と三好氏の接点が明確化
赤沢長経が阿波細川家の細川澄元に仕え、その系統から後に赤沢宗伝（板西城主）が三好実休に仕えた。天文年中の頼経が三好氏と関係があるのは、この畿内残留組の系統と推定される。

### 42. 赤沢常親（あかざわ つねちか）

#### 再調査結果

直接的な個別史料は今回も確認できなかった。ただし、以下の推定が可能である。

**通称「彦三郎」の分析:**
- 「彦」は武家の通称に多く用いられる美称。「三郎」は三男を意味する。
- 経光の嫡子は武経（太郎）であるため、常親は経光の**三男**と推定される。
- 「常」の字は祖父・常興からの偏諱（へんき＝目上の人物から一字を授かること）であり、常興系の正統な一族であることを示す。

**活動時期の推定:**
- 父・経光が14世紀中頃〜後半に活動、弟の武経が明徳の乱（1391年）に参戦していることから、常親も同時期に活動したと推定。
- 大塔合戦（1400年）の時期に存命であった可能性がある。

**系図上の位置:**
- 小笠原諸流系図では、経光の子として本流（武経→満経→経隆→朝経）とは別系統に位置する庶流。
- 嫡子ではないため、独自の所領を持って分家した可能性がある。

### 43. 赤沢常能（あかざわ つねよし）

| 項目 | 内容 |
|------|------|
| 通称 | 情報なし |
| 官位 | 治部少輔（じぶのしょう） |
| 生没年 | 早世（若くして死去） |
| 父 | 経光（推定） |
| 位置づけ | 経光の庶子 |

#### 再調査結果

個別の史料は確認できなかったが、以下の分析が可能である。

**「治部少輔」の意義:**
- 治部省は、朝廷における氏族の管理・外交・仏事を所管する部署。
- 「少輔」は治部省の次官（従五位下相当）。
- 赤沢氏の中で「治部少輔」を名乗った人物は複数おり（経光自身も治部少輔を称した）、経光から常能への官位継承を示す。
- 経光が「治部少輔・伊豆守」を兼ねていたとされ、常能は経光から「治部少輔」の官位を受け継いだ可能性が高い。これは常能が**一時は嫡子候補であった**ことを示唆する。

**「早世」の意味:**
- 早世とは成人前後に死去したことを意味する。通常10代〜20代前半での死去。
- 治部少輔の官位を持ちながら早世したことは、一族の中で将来を嘱望されていたにもかかわらず、病死または戦死した可能性を示す。
- 常能が早世したために、嫡子の座が武経に移った可能性がある（推定）。

**「常」の偏諱:**
- 常親と同じく、「常」は祖父・常興からの偏諱。

#### 再調査結果

個別の史料は確認できなかったが、以下の分析が可能である。

**通称「左京」の分析:**
- 「左京」は左京職（朝廷の京都左半分を管轄する官庁）に由来する武家の通称。
- 中流〜上流の武家に多く用いられた通称であり、赤沢氏の中では比較的高い格式を持つ。

**「経」の偏諱:**
- 「経」の字は赤沢氏本流に代々受け継がれる通字（つうじ＝一族で代々使う漢字）。
- 清経→安経→経顕→氏経→常興→経光→武経→満経→経隆→朝経と、「経」の字が一貫して使われている。
- 経家が「経」の字を持つことは、本流に近い人物であることを示す。

**推定される立場:**
- 「常」の字を持つ常親・常能とは異なり、「経」の字を持つことから、経光の子の中でも本流に近い位置づけであった可能性がある。
- 嫡子・武経に次ぐ立場であったかもしれない。

#### 再調査結果

#### 「八刺」について（重要な発見）

今回の調査でも「八刺」の正確な意味を特定する決定的な史料は見つからなかったが、以下の複数の可能性を提示する。

**仮説1: 地名由来**
- 「八刺」が信濃国内の小字（こあざ＝小地名）に由来する可能性。
- 中世武家は、所領の地名を通称・苗字とすることが一般的であった。
- 信濃国更級郡・埴科郡周辺に「八刺」に該当する地名が存在した可能性があるが、現存する地名としては確認できなかった。

**仮説2: 武勇に関連する異名**
- 「刺」は武器（槍・刀）で「刺す」ことを意味し、「八刺」は「八方を刺す」＝「全方位を攻める猛者」という武勇を称える異名の可能性。
- 中世の武士には武勇にちなむ異名が付けられることがあった（例: 朝経の「悪太郎」）。

**仮説3: 読みは「やさし」**
- 「八刺」を「やさし」と読む可能性。信濃方言や中世の読みにおいて、このような読み方が存在した可能性がある。
- ただし、確証となる史料は未発見。

**通称「左衛門」の分析:**
- 左衛門府（朝廷の護衛機関）に由来する通称。武家で最も一般的な通称の一つ。

**「光景」の名の分析:**
- 「光」の字は赤沢氏では珍しい。経光の「光」を受け継いでいる可能性があり、経光から特に寵愛された子であった可能性を示唆する。
- なお、小笠原諸流系図の一部では「経光」の系図位置に「光景」を記載するものがあるとの情報もあるが（※要追加検証）、これは同一人物の別名なのか、誤写なのか、判然としない。

### 52. 赤沢秀國（あかざわ ひでくに）【重要な新発見あり】

| 項目 | 内容 |
|------|------|
| 通称 | 情報なし |
| 官位 | 対馬守（つしまのかみ） |
| 役職 | 四宮の頭（しのみやのかしら） |
| 生没年 | 不明（15世紀前半に活動と推定） |
| 居城 | 塩崎城（信濃国更級郡） |
| 位置づけ | 赤沢氏本流または有力庶流 |

#### 再調査結果（新発見を含む）

##### 大塔合戦（1400年）への参戦

**今回の調査で最も重要な発見**として、応永7年（1400年）の大塔合戦に「赤沢秀国」が**塩崎城主**として参戦していたことが確認された。

- 大塔合戦は、信濃守護に任じられた小笠原長秀が、村上氏・井上氏・高梨氏・仁科氏ら有力国人領主の連合軍（大文字一揆）と善光寺平南部で争った合戦。
- 小笠原長秀は、横田城での敗戦後、一族の赤沢秀国の居城である**塩崎城**への撤退を試みた。
- 塩崎城は「天然の要害」であり、峠を越えれば小笠原氏の本拠・松本（信濃府中）に近い戦略的拠点であった。
- 長秀は塩崎城に辛うじて到達し（わずか150騎）、大井光矩の調停で和議が成立して京都に撤退した。

この記録から、**秀國は15世紀初頭に塩崎城を統治していた赤沢氏の有力者**であったことが明らかになった。

##### 「四宮の頭」の意味（解明）

**「四宮」の由来:**
- 武水別神社（たけみずわけじんじゃ）は、長野県千曲市八幡に鎮座する古社で、延喜式内社（名神大社）。
- 信濃国には「一宮」諏訪大社から順に宮の格付けがあり、武水別神社は**信濃国四宮**（四番目の宮）とされる。
- 武水別神社の神領（荘園）を**四宮荘**と呼ぶ。

**四宮荘と赤沢氏の関係:**
- 建武2年（1335年）の中先代の乱において、武水別神社の神官家であった**四宮氏**（四宮左衛門太郎）が北条時行方として船山守護所を襲撃（青沼合戦）。
- 四宮氏は鎮圧されて滅亡し、武水別神社の社殿も焼失した。
- この乱の鎮圧に功績があった赤沢常興（小笠原貞宗に従軍）が、恩賞として四宮荘北条（塩崎）の地頭職を与えられた。
- 以後、赤沢氏が四宮荘の実質的な支配者となった。

**「頭」の意味:**
- 武水別神社には「**頭人行事**」（とうにんぎょうじ）と呼ばれる神事がある。
- 「頭人」とは、氏子の中から選出されて祭祀を主宰する最高位の役職。
- 「四宮の頭」とは、四宮荘の**頭人**＝**四宮荘の祭祀と行政の長**を意味する。
- これは単なる軍事的な領主ではなく、**武水別神社の神領の管理と祭祀の主宰者**という宗教的・行政的な地位を示す。

**歴史的意義:**
- 赤沢秀國が「対馬守」の官位と「四宮の頭」の役職を持っていたことは、赤沢氏が信濃において**軍事的**（塩崎城主）・**宗教的**（四宮荘の祭祀管理者）・**行政的**（地頭職）な三重の権力を持つ有力国人であったことを示す。
- 大塔合戦（1400年）において、信濃守護・小笠原長秀が最終的に秀國の塩崎城に逃れたことは、赤沢氏が小笠原氏にとって最も信頼できる一族であったことの証左である。

### 56. 赤沢経康（あかざわ つねやす）

|------|------|

| 官位 | 伊豆守（いずのかみ） |
| 活動時期 | 天文年中（1532〜1555年） |
| 生没年 | 不明 |
| 位置づけ | 赤沢氏本流に近い人物 |

#### 再調査結果

個別の史料は確認できなかったが、以下の重要な推定が可能である。

**「伊豆守」の意義:**
- 「伊豆守」は赤沢氏の初代・清経以来、代々の本流当主が継承してきた官位。
- 天文年中に「伊豆守」を称していることは、経康が**赤沢氏本流の当主またはそれに準ずる立場**であったことを示す。

**政経・経智との関係:**
- 赤沢政経（伊豆守、1467〜1512年）は永正6年（1509年）に信濃帰還し、永正9年（1512年）に飯沼の館で戦死。
- 政経の子・経智（式部少輔、1511年生）は天文年間に塩崎城で小笠原長時に仕えた。
- 経康が天文年中に「伊豆守」を称しているとすると、経智（式部少輔）とは**別系統の伊豆守**、すなわち信濃に残留した赤沢氏の別の嫡流を継ぐ人物であった可能性がある。

**武田信玄との関係:**
- 天文年間（1532〜1555年）は、武田信玄が信濃侵攻を本格化した時期と重なる。
- 天文14年（1545年）以降、赤沢経智は小笠原長時に従って武田軍と戦った。
- 経康が経智と同時期に「伊豆守」を称していたとすると、一族内で本流と分家が並立し、それぞれ異なる立場で武田の侵攻に対応した可能性がある。

**武田分限帳との関連:**
- 武田分限帳（永禄12年〜元亀4年頃）に「赤沢」が**信州先方衆**として40騎の軍役で記録され、館は**松本市浅間温泉**にあった。
- 経智は小笠原長時に従って越後に逃亡したが、経康の系統は**武田に降伏して信濃に残留**し、武田家臣となった可能性がある。
- これが武田分限帳に記録された「赤沢」の正体かもしれない。

### 57. 赤沢頼経（あかざわ よりつね）【新たな推定あり】

| 項目 | 内容 |
|------|------|
| 通称 | 情報なし |
| 官位 | 情報なし |
| 活動時期 | 天文年中（1532〜1555年） |
| 関連 | 三好氏 |
| 位置づけ | 畿内残留組の赤沢氏 |

#### 再調査結果

個別の史料は確認できなかったが、三好氏との関係について重要な推定が可能となった。

**赤沢氏と三好氏の関係（系譜の整理）:**

赤沢氏が三好氏と関係を持つに至った経緯は以下の通り。

1. **赤沢朝経**（1451〜1507年）が細川政元の内衆として畿内で活動。
2. **赤沢長経**（朝経の弟/養子）が朝経死後、阿波細川家の**細川澄元**に仕えた。
3. 長経は永正5〜6年（1508〜1509年）に河内で斬首されて死去。
4. 長経の系統（または名跡）は阿波に残り、後に**三好氏**と接触。
5. 天文年間、赤沢氏の一部が三好長慶の勢力圏で活動。
6. 後に小笠原宗伝が三好実休の婿となり、**板西城主・赤沢信濃守宗伝**と称した。

**頼経の位置づけ（推定）:**
- 頼経が「天文年中・三好氏」と記録されていることから、畿内に残留した赤沢氏（長経系統）の一族と推定される。
- 細川澄元→三好元長→三好長慶と続く三好氏の台頭の中で、赤沢氏の一部がその配下に組み込まれていったと考えられる。
- 頼経は、信濃の赤沢本流と畿内の三好氏をつなぐ**中間世代の人物**であった可能性がある。

**三好長慶と赤沢氏の接点:**
- 弘治元年（1555年）、小笠原長時・長勝・貞経は三好長慶を頼って上洛した。
- 三好長慶は小笠原庶流の赤沢氏にとって、既に畿内に残留していた一族を通じて関係があった。
- 頼経が天文年間にこの畿内の赤沢ネットワークの中で活動していた可能性は高い。

### 58. 赤沢経興（あかざわ つねおき）

| 項目 | 内容 |
|------|------|
| 通称 | 情報なし |
| 官位 | 情報なし |
| 活動時期 | 天文年中（1532〜1555年） |
| 生没年 | 不明 |
| 位置づけ | 赤沢氏の一族 |

#### 再調査結果

個別の史料は確認できなかった。

**「経」の通字:**
- 経興は赤沢氏の通字「経」を持つ。本流に近い人物と推定される。

**天文年間の赤沢氏の状況:**
天文年間（1532〜1555年）の赤沢氏は、以下の複数の系統に分かれて活動していた。

1. **信濃本流（経智系）**: 小笠原長時に仕え、塩崎城を拠点に武田と対峙。天文19年（1550年）に越後に逃亡。
2. **信濃残留組（経康系？）**: 武田に降伏し、信州先方衆として40騎の軍役を務めた。
3. **畿内残留組（頼経系？）**: 三好氏に仕え、阿波・畿内で活動。
4. **信濃土着組（信基・清純系）**: 信州に土着して存続。

経興がこの4系統のいずれに属するかは不明だが、「天文年中」の活動記録があることから、上記のいずれかの系統に属していたと考えられる。

**推定:**
- 経興が経康（伊豆守）や経智（式部少輔）と同時期に活動していることから、同世代の一族であり、信濃における赤沢氏の勢力を支えた人物の一人であった可能性が高い。

### 59. 赤沢信基（あかざわ のぶもと）

| 項目 | 内容 |
|------|------|
| 通称 | 情報なし |
| 官位 | 武部少輔（たけべのしょう） |
| 生没年 | 不明 |
| 子孫 | 信州にあり |
| 位置づけ | 信濃に土着した赤沢氏の庶流 |

#### 再調査結果

個別の史料は確認できなかったが、以下の重要な推定が可能である。

**「武部少輔」の意義:**
- 「武部」は「兵部」（ひょうぶ）の別表記であると考えられる。兵部省は軍事を所管する部署。
- ただし、「武部」という表記は一般的ではなく、赤沢氏独自の表記である可能性がある。
- 清純（No.60）と同じ「武部少輔」を称していることは、同一系統の分家であることを強く示唆する。

**「信」の字について:**
- 「信」は赤沢氏の通字ではない。
- 「信濃」の「信」に由来する可能性がある（信濃に根を下ろした一族としてのアイデンティティ）。
- あるいは、武田信玄の「信」からの偏諱の可能性もある。武田に降伏した後に「信」の字を授けられたとすれば、武田分限帳の赤沢氏（信州先方衆、40騎）との関連が浮上する。

**「子孫は信州にあり」の意味:**
- これは系図の注記として記されたもので、系図作成時点（江戸時代と推定）において、信基の子孫が信州に存在していたことを意味する。
- 赤沢氏本流（経智→貞経系統）は越後に逃亡し、その後徳川に仕えて江戸に移住した。
- これに対し、信基の系統は信濃に残り続け、**武田→徳川（松本藩など）の支配下で存続**した可能性がある。

**武田分限帳との関連:**
- 武田分限帳に記録された赤沢氏（信州先方衆、40騎、浅間温泉に館）は、信基または清純の系統である可能性がある。
- 浅間温泉は松本市東部に位置し、小笠原氏の本拠・松本に近い。赤沢氏が小笠原氏の庶流として松本周辺に所領を持っていたことと整合する。

### 60. 赤沢清純（あかざわ きよずみ）

| 項目 | 内容 |
|------|------|
| 通称 | 情報なし |
| 官位 | 武部少輔（たけべのしょう） |
| 生没年 | 不明 |
| 子孫 | 信州にあり |
| 位置づけ | 信濃に土着した赤沢氏の庶流 |

#### 再調査結果

個別の史料は確認できなかったが、信基と同様の分析が可能である。

**信基との関係:**
- 同じ「武部少輔」を称し、同じく「子孫は信州にあり」と記録されている。
- 信基の子または弟と推定される。同じ系統の人物であることはほぼ確実。

**「清」の字について:**
- 「清」は赤沢氏の初代・**清経**に由来する可能性がある。
- 初代の名を一字受け継ぐことは、赤沢氏の正統性を主張する意味がある。

**信基・清純系統の歴史的位置づけ:**
- 赤沢氏本流が畿内進出（朝経）→越後逃亡（経智）→徳川仕官（貞経）と移動したのに対し、信基・清純の系統は**信濃に留まり続けた唯一の赤沢氏**であった可能性がある。
- 「子孫は信州にあり」という注記は、江戸時代に系図が作成された際に、本流（幕臣となった貞経系統）が「我々は江戸にいるが、信基・清純の子孫は今も信州にいる」と記録したものと推定される。
- この系統が現在の長野県に存在する赤沢姓の一部の祖先である可能性がある。

### 武田分限帳の赤沢氏

| 記録時期 | 永禄12年〜元亀4年頃（1569〜1573年頃） |

- 赤沢経智は天文19年（1550年）に越後に逃亡しているため、武田分限帳の「赤沢」は経智とは**別系統**の赤沢氏。
- 経康（伊豆守・天文年中）の系統、または信基・清純の系統が武田に降伏して「信州先方衆」となった可能性が高い。
- 40騎は相当な軍事力であり、赤沢氏が信濃においてなお有力な国人であったことを示す。

### 四宮氏の滅亡と赤沢氏の四宮荘支配

- 建武2年（1335年）の中先代の乱で、武水別神社の神官家・四宮氏が北条時行方として蜂起し、鎮圧されて滅亡。
- 赤沢常興が小笠原貞宗に従って乱の鎮圧に貢献し、四宮荘北条（塩崎）の地頭職を獲得。
- 以後、赤沢氏が四宮荘の実質的支配者となり、秀國の代に「四宮の頭」として祭祀・行政の長を務めた。
- この「四宮の頭」の称号は、赤沢氏が単なる武家領主ではなく、**神領の管理者**という宗教的権威も持っていたことを示す。

### 赤沢氏と三好氏の関係（系譜の整理）

| 時期 | 人物 | 関係 |
|------|------|------|
| 1491年〜1507年 | 赤沢朝経 | 細川政元の内衆 |

| 天文年中 | 赤沢頼経 | 三好氏と関係 |
| 1555年頃 | 小笠原長時 | 三好長慶を頼って上洛 |
| 戦国後期 | 赤沢宗伝 | 三好実休の家臣、板西城主 |

- 頼経はこの過渡期の人物であり、長経系統と三好氏をつなぐ存在であったと推定される。
