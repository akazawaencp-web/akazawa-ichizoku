## 1. 清和天皇（せいわてんのう）

### 基本情報

| 項目 | 内容 |
|------|------|
| 生没年 | 嘉祥3年3月25日（850年5月10日）〜 元慶4年12月4日（881年1月7日） |
| 享年 | 31歳 |
| 諱 | 惟仁（これひと） |
| 代数 | 第56代天皇 |
| 在位期間 | 天安2年（858年）〜 貞観18年（876年） |

### 官位・役職

天皇（第56代）。わずか9ヶ月で皇太子となり、9歳で即位。

### 家族関係

| 関係 | 人物 |
|------|------|
| 父 | 文徳天皇（第55代） |
| 母 | 藤原明子（ふじわらのあきらけいこ）── 太政大臣・藤原良房の娘 |
| 妻 | 藤原高子（二条后）ほか多数 |
| 子 | 陽成天皇（第一皇子）、貞純親王（第六皇子）ほか多数の皇子・皇女 |

### 主な事績

- **幼年即位**: 9歳で即位し、祖父の藤原良房が外戚として実権を掌握
- **応天門の変（866年）**: 応天門放火事件を機に、藤原良房が正式に人臣初の摂政に就任。藤原北家による摂関政治が確立される転機となった
- **若年譲位**: 貞観18年（876年）、27歳で第一皇子の陽成天皇に譲位し太上天皇となる
- **出家と修行**: 元慶3年（879年）5月に出家。翌年3月より丹波国水尾の地に入り、絶食を伴う激しい苦行を行った
- **崩御**: 病により粟田山荘の円覚寺に移り、31歳で崩御

### 人物像・逸話

- 幼少即位のため、実質的な政治権力は祖父・藤原良房に握られていた
- 譲位後は仏門に入り、近畿各地を巡りながら激しい修行に励んだ。水尾の里人は天皇を敬い、新しい仏像を建てて迎えた
- 出家後わずか2年で病に倒れ、短い生涯を閉じた

### 墓所・ゆかりの地

- **水尾山陵**（みずのおやまのみささぎ）── 京都府京都市右京区嵯峨水尾清和
- 遺骨は洛東の粟田山で火葬され、遺言により水尾山上に埋葬
- **清和天皇社** ── 水尾の里に鎮座する天皇を祀る社

### 歴史的意義

- 子孫が臣籍降下して「源」姓を賜り、後の清和源氏となる
- 源頼朝、源義経、足利尊氏、武田信玄など、日本史を動かした武将の始祖
- 藤原摂関政治確立の画期となった天皇

### 赤沢氏との関係性

赤沢氏は清和源氏の流れを汲む小笠原氏の庶流であり、清和天皇は赤沢氏の最も遠い直系祖先にあたる。

### 後世の評価

幼くして即位し、藤原氏の影響下で政治を行ったが、武家の棟梁を輩出する清和源氏の始祖として、後世に「武家の血統の源流」として特別な存在となった。

## 2. 貞純親王（さだずみしんのう）

### 基本情報

| 項目 | 内容 |
|------|------|
| 生没年 | 貞観15年（873年）〜 延喜16年5月7日（916年6月10日） |
| 享年 | 44歳 |
| 別称 | 桃園親王（とうえんしんのう） |

### 官位・役職

- 四品（親王の位階）
- 上総太守・上野太守（親王任国）
- 中務卿
- 兵部卿

### 家族関係

| 関係 | 人物 |
|------|------|
| 父 | 清和天皇（第56代） |
| 母 | 棟貞王の娘 |
| 子 | 源経基（六孫王）、経生王 |

### 主な事績

- 貞観15年（873年）、清和天皇の皇子7名・皇女4名とともに親王宣下を受ける
- 宇多朝にて上総・上野の太守を歴任
- 中務卿・兵部卿を歴任するも、位階は四品に留まった

### 人物像・逸話

- **桃園親王の由来**: 当時の人々の夢に、一条大宮の桃園池に住む七尺の竜として現れたことから「桃園親王」と号した
- 政治的には大きな活躍の記録はないが、子の経基・経生が源姓を賜り臣籍降下したことで、清和源氏の始祖的存在となった

### 墓所・ゆかりの地

情報なし（具体的な墓所の記録は確認できず）

### 歴史的意義

清和天皇の血統を武家の世界へ繋いだ要の人物。子の源経基が臣籍降下して「源」姓を名乗り、清和源氏の始まりとなった。

### 赤沢氏との関係性

赤沢氏の直系祖先。貞純親王の子・源経基が清和源氏の祖となり、その子孫が赤沢氏へと分岐していく。

### 後世の評価

清和源氏の系図において、天皇家から武家への橋渡し役として記憶されている。

## 3. 源経基（みなもとのつねもと）── 六孫王

### 基本情報

| 項目 | 内容 |
|------|------|
| 生没年 | 寛平年間（889〜898年）頃 ※推定 〜 応和元年（961年） ※諸説あり |
| 享年 | 45歳（『尊卑分脈』による） ※推定 |
| 別称 | 六孫王（ろくそんのう） |

※生没年には諸説あり。『尊卑分脈』は応和元年（961年）45歳没とするが、長男・源満仲が延喜12年（912年）頃の生まれとすると矛盾が生じる。

### 官位・役職

- 武蔵介
- 鎮守府将軍
- 征東副将軍
- 追捕使次官
- 大宰権少弐
- 位階: 正四位上

### 家族関係

| 関係 | 人物 |
|------|------|
| 父 | 貞純親王（清和天皇第六皇子） |
| 母 | 源能有の娘 |
| 子 | 源満仲、源満政、源満実、源満季 ほか |

### 主な事績

- **臣籍降下**: 源姓を賜り、皇族から臣下となる。清和源氏の祖
- **平将門の乱（939年）**: 武蔵介として赴任中に将門の謀反を朝廷に報告。当初は虚偽と疑われ拘禁されたが、将門が実際に挙兵すると釈放され、征東副将軍に任じられた
- **藤原純友の乱（941年）**: 追捕使次官として純友の残党掃討に参加。豊後国で桑原生行を捕縛する軍功を挙げた

### 人物像・逸話

- **『将門記』での評価**: 「悪人の手口を身につけ、盗賊の心を持っている」と酷評されている。武蔵での武芝との紛争では逃亡したと記されている
- **武功の誇示**: 純友の乱で捕虜の首を自ら京都に持ち帰り武勲を誇示した
- 軍事的才能には疑問が呈される一方、朝廷への告発や情報戦には長けていた
- 関東には悪評しか残せなかったとの評価もある

### 墓所・ゆかりの地

- **六孫王神社**（京都府京都市南区）── 経基の邸宅跡に子の満仲が創建。清和源氏発祥の宮として祀られる

### 歴史的意義

- 清和源氏の初代。皇族から武家への転換点となった人物
- 承平天慶の乱（将門の乱・純友の乱）に関与し、源氏が武家として歩み始める契機を作った
- 子の満仲以降、清和源氏は武門の棟梁へと発展していく

### 赤沢氏との関係性

赤沢氏の直系祖先。源経基の臣籍降下により「源」姓が始まり、その血統が満仲→頼信→頼義→義家→義光→（甲斐源氏）→小笠原氏→赤沢氏と繋がる。

### 後世の評価

- 同時代の評価は必ずしも高くなかったが、清和源氏の始祖として後世に神格化された
- 六孫王神社は源氏ゆかりの社として現在も信仰を集める

## 4. 源満仲（みなもとのみつなか）── 多田満仲

### 基本情報

| 項目 | 内容 |
|------|------|
| 生没年 | 延長10年（912年）頃 〜 長徳3年8月27日（997年9月29日） |
| 享年 | 87歳前後 ※推定 |
| 別称 | 多田満仲（ただまんじゅう） |
| 法名 | 満慶 |

### 官位・役職

- 鎮守府将軍
- 摂津守
- 武蔵守、上総介、常陸介、越後守、越前守、下野守、美濃守、信濃守、伊予守、陸奥守 ほか多数の国司を歴任
- 検非違使

### 家族関係

| 関係 | 人物 |
|------|------|
| 父 | 源経基（六孫王） |
| 母 | 情報なし |
| 子（男子） | 源頼光（長男）、源頼親（次男）、源頼信（三男）、源頼平（四男）、源頼明（五男）、源頼貞（六男）、源頼範（七男）、源賢（僧侶） ほか |

### 主な事績

- **安和の変（969年）**: 藤原氏の政争に関与し、反対勢力の有力者の逮捕に貢献。正五位下を得る
- **武士団の形成**: 摂津国多田庄（現・兵庫県川西市多田）に武士団を組織。清和源氏最初の本拠地となった
- **藤原摂関家との結びつき**: 花山天皇退位事件（寛和の変、986年）では、藤原道兼を警護したとされる
- **出家**: 永延元年（987年）多田の邸宅にて出家し、法名を満慶と称した

### 人物像・逸話

- 48歳という遅咲きで歴史に登場（天徳4年・960年、平将門の子の捜索が初見）
- 政治と武力の両面に長け、藤原摂関家に取り入ることで清和源氏の基盤を築いた
- 「官職に限定されずに武士団を用いた最初の例」とされる
- 息子の源賢（僧侶）から仏法の教えを受けて出家し、天寿を全うした

### 墓所・ゆかりの地

- **多田神社**（兵庫県川西市多田）── 満仲を主祭神として祀る。遺骸はここに葬られた
- 高野山にも供養塔がある
- 源氏三神社のひとつ

### 歴史的意義

- 清和源氏を武門として確立した最大の功労者
- 摂津国多田を拠点に武士団を形成し、以後の源氏の軍事基盤を作った
- 長男・頼光は摂津源氏、次男・頼親は大和源氏、三男・頼信は河内源氏の祖となり、清和源氏の三大分流を生み出した

### 赤沢氏との関係性

赤沢氏の直系祖先。満仲の三男・頼信の系統が河内源氏となり、その子孫から甲斐源氏・小笠原氏・赤沢氏が分かれていく。

### 後世の評価

- 武門・源氏の礎を築いた人物として高く評価される
- 多田神社は「源氏発祥の地」として現在も参拝者が絶えない
- 天禄元年（970年）創建の多田神社は源氏三神社のひとつ

## 5. 源頼光（みなもとのよりみつ）

### 基本情報

| 項目 | 内容 |
|------|------|
| 生没年 | 天暦2年（948年）〜 治安元年7月19日（1021年8月28日） |
| 享年 | 74歳 |
| 別称 | 源頼光（らいこう）とも読む |

### 官位・役職

- 従四位下
- 摂津守
- 美濃守、但馬守、伊予守、備前守
- 内蔵頭
- 大内守護

### 家族関係

| 関係 | 人物 |
|------|------|
| 父 | 源満仲 |
| 母 | 源俊の娘（嵯峨源氏） |
| 妻 | 藤原元平の娘、平惟仲の養女、慶滋保章の娘 ほか |
| 兄弟 | 頼親（次男）、頼信（三男）、頼平（四男）、頼明（五男）、頼貞（六男）、頼範（七男） |
| 子 | 源頼国（嫡男）、源頼家、源頼基 ほか |

### 主な事績

- **摂津源氏の祖**: 父・満仲から摂津国多田の地盤を相続し、摂津源氏を開いた
- **藤原道長への近侍**: 兄弟の中で最も道長に近く、その財政的支援者でもあった
- **大江山夷賊追討**: 寛仁2年（1018年）、一条天皇の勅命を受け、頼光四天王らと共に大江山の夷賊討伐を実施
- **酒呑童子退治伝説**: 大江山の鬼・酒呑童子を毒酒で酔わせて退治したという伝説は後世に広く語り継がれた
- **土蜘蛛退治**: 病床にあった頼光が妖怪・土蜘蛛を退治した説話も有名

### 人物像・逸話

- **頼光四天王**: 渡辺綱（鬼の腕を斬った伝説）、坂田金時（金太郎のモデル）、碓井貞光、卜部季武の4人の勇士を従えた
- **童子切安綱**: 酒呑童子を斬った名刀は「天下五剣」のひとつとして現存（国宝・東京国立博物館蔵）
- 武勇だけでなく、政治的手腕と財力にも優れ、藤原道長の有力な後援者であった

### 墓所・ゆかりの地

- **多田神社**（兵庫県川西市）── 本殿真裏に「御神廟」として父・満仲とともに祀られる（禁足地）
- **源頼光朝臣塚**（京都市）

### 歴史的意義

- 摂津源氏の祖として、平安京における源氏の地位を確立
- 酒呑童子退治の伝説は能・歌舞伎・浮世絵など日本文化に多大な影響を与えた
- 鬼退治伝説は、実際には山賊や反乱勢力の討伐を物語化したものと推定される

### 赤沢氏との関係性

赤沢氏の直系ではないが、赤沢氏の祖である源頼信（河内源氏の祖）の兄にあたる。赤沢氏にとっては大伯父の系統。

### 後世の評価

- 酒呑童子退治の英雄として、日本の伝説・文芸に不朽の名を残す
- 頼光四天王の物語は能「大江山」「土蜘蛛」など多数の芸能作品の題材となった

| 享年 | 80歳 |

### 官位・役職

- 河内守（1046年任官）
- 大和守、甲斐守
- 従四位上（1047年叙位）
- 常陸介、上総介、美濃守 ほか

### 家族関係

| 関係 | 人物 |
|------|------|
| 父 | 源満仲 |
| 母 | 情報なし |
| 妻 | 修理命婦 |
| 兄 | 源頼光（長男）、源頼親（次男） |
| 子 | 源頼義（嫡男）、源頼清、源頼季（井上氏の祖）、源頼任、源義政 ほか |

### 主な事績

- **河内源氏の祖**: 河内国石川郡壺井（現・大阪府羽曳野市）に拠点を築き、河内源氏を開いた
- **平忠常の乱鎮圧（1028〜1031年）**: 関東で3年間続いた反乱を、赴任の知らせだけで忠常が戦わずして降伏させた。これにより河内源氏の東国における威信が確立
- **道長四天王**: 藤原道長の武力的な側近として、平維衡・平致頼・藤原保昌とともに「道長四天王」と称された
- **東国経営**: 兄・頼光が京都で藤原氏に仕えたのに対し、頼信は東国に基盤を築いた

### 人物像・逸話

- 平忠常の乱では、「頼信が来る」という噂だけで敵が降伏したという逸話は、頼信の武名がいかに恐れられていたかを物語る
- 兄・頼光が京の政治的パトロンであったのに対し、頼信は東国の武士団統率者として独自の道を歩んだ

### 墓所・ゆかりの地

- **通法寺跡**（大阪府羽曳野市）── 頼義・義家とともに墓がある
- **壺井八幡宮**（大阪府羽曳野市）── 河内源氏の氏神

### 歴史的意義

- 河内源氏の祖として、源頼朝・源義経・足利尊氏など日本史上最も重要な武将群を輩出する系統を開いた
- 東国武士団との結びつきを確立し、後の鎌倉幕府の基盤を準備した

### 赤沢氏との関係性

赤沢氏の直系祖先。頼信→頼義→義光→（甲斐源氏）→小笠原氏→赤沢氏の系譜。

### 後世の評価

- 鎌倉幕府・室町幕府の将軍家の祖として、武家政権の源流と位置づけられる
- 「名声だけで敵を降伏させた」逸話は、理想的な武将像として語り継がれた

## 7. 源頼親（みなもとのよりちか）

### 基本情報

| 項目 | 内容 |

```
                      └─ 小笠原長経（嫡男）
                           └─ 赤沢清経（赤沢氏初代）
                                ── 伊豆国田方郡赤沢郷を本貫として赤沢を称す
```

### 後世の評価

- 笙の名手として文武両道の武将像が伝えられる
- 兄のために官職を捨てた義光の行動は、武家の兄弟愛の理想として語り継がれた
- 足柄峠での秘曲伝授の説話は、能・歌舞伎・浮世絵の題材となった
- 甲斐源氏の祖として、武田氏・小笠原氏・赤沢氏など多くの子孫に敬われている

## 赤沢氏と清和源氏の関係まとめ

### 赤沢氏の出自

赤沢氏は **清和源氏 → 河内源氏 → 甲斐源氏 → 小笠原氏** の庶流である。

### 系譜の全体像

| 世代 | 人物 | 備考 |
|------|------|------|
| 1代 | 清和天皇 | 第56代天皇 |
| 2代 | 貞純親王 | 第六皇子 |
| 3代 | 源経基（六孫王） | 臣籍降下、清和源氏の祖 |
| 4代 | 源満仲 | 多田源氏の祖 |
| 5代 | 源頼信 | 河内源氏の祖 |
| 6代 | 源頼義 | 前九年の役 |
| 7代 | 源義光（新羅三郎） | 甲斐源氏の祖 |
| 8代 | 源義清 | 甲斐に移住 |
| 9代 | 源清光 | 甲斐源氏を拡大 |
| 10代 | 加賀美遠光 | |
| 11代 | 小笠原長清 | 小笠原氏初代 |
| 12代 | 小笠原長経 | |
| 13代 | 赤沢清経 | 赤沢氏初代 |

### 赤沢氏の発祥地

- **本貫地**: 伊豆国田方郡赤沢郷（現・静岡県伊東市赤沢）
- 初代・清経が伊豆から信濃国埴科郡埴生（現・長野県千曲市）に移住
- 以後、子孫は主に信濃を本拠とした

### 赤沢氏の主要系図

『寛政重修諸家譜』『小笠原諸流系図』による:

```
長経 ─ 清経（赤沢源次郎）─ 安経 ─ 経顕 ─ 氏経 ─ 常興 ─ 経光
─ 武経 ─ 満経 ─ 経隆 ─ 朝経 ─ 政経 ─ 経智 ─ 貞経（家康・秀忠に仕え、小笠原に復姓）
```

### 赤沢氏の主な活動

- 明徳2年（1391年）: 赤沢武経が小笠原長基に従い出陣、足利義満から感状を受領
- 永享12年（1440年）: 結城合戦に参加
- 戦国期: 細川政元の重臣として朝経が活躍
- 永正期: 細川政元暗殺後、朝経は敗死、長経は処刑
- 慶長9年: 貞経が徳川家康に召し出され、小笠原に復姓。江戸幕府旗本に
- 子孫は小笠原流弓馬術礼法宗家となる
