### 62. 赤沢朝治（あかざわ ともはる）

| 項目 | 内容 |
|------|------|
| 氏名 | 赤沢朝治 |
| 父 | 赤沢政経の子か（※推定） |
| 生没年 | 不明 |
| 活動時期 | 永正〜大永年間（1500年代前半）と推定 |

#### 再調査結果

赤沢政経の子として確認できるのは、現時点では**赤沢経智**（永正8年/1511年生）のみである。Wikipediaの赤沢政経・赤沢経智の項目、姓氏家系メモの赤沢氏系図、いずれにおいても「朝治」の名は確認できなかった。

**考察と推定**：
- 赤沢氏の主要系図（長経→清経→...→朝経→政経→経智→貞経）に「朝治」は含まれていない
- 「朝」の字は赤沢朝経（政経の父）から受け継いだ通字であり、赤沢氏嫡流の命名法則に合致する
- 政経には経智以外にも子がいた可能性があり、朝治は経智の兄弟（庶子含む）と推定される
- 政経は永正9年（1512年）に飯沼館で討死しており、朝治が政経の子であれば、信濃で武田氏の勢力拡大期に活動した人物と考えられる
- **「朝」の字を持つことから、嫡流に近い立場の人物だった可能性が高い**
- あるいは、政経の兄弟（朝経の子で政経以外の者）の可能性も完全には排除できない

**結論**：一次史料（『寛政重修諸家譜』等）での確認が不可欠。Web上の公開情報では依然として特定困難。

### 64. 赤沢貞清（あかざわ さだきよ）— 赤沢太郎・兵庫頭

| 項目 | 内容 |
|------|------|
| 氏名 | 赤沢貞清 |
| 通称 | 赤沢太郎 |
| 官位 | 兵庫頭（ひょうごのかみ） |
| 子 | 宗春（左馬助） |
| 活動時期 | 室町後期〜戦国初期（15世紀後半〜16世紀前半）と推定 |

#### 再調査結果

**岡山県立図書館（デジタル岡山大百科）** に所蔵の論文「**赤沢兵庫助政定の一門と備中赤沢氏―細川典厩家・野州家との関わりを中心に―**」が、貞清と政定の関係を解明する鍵となる可能性がある。ただし、論文本文へのオンラインアクセスはシステム改修中により不可であった。

**新たに判明した情報・推定**：

1. **「兵庫」の通字**：備中赤沢氏では「兵庫」が代々使われている
   - 貞清 = 兵庫頭
   - 政定 = 兵庫助
   - 宗照 = 兵庫助
   - 宗栄 = 兵庫助
   - 赤沢兵庫頭政信 = 天正12年（1584年）に柏島八幡宮を創建

2. **貞清と政定の関係**：
   - 両者とも「兵庫」を称していることから、同一人物か、極めて近い血縁関係にある
   - 貞清が通称「赤沢太郎」であることは、赤沢氏の嫡流（長男筋）を意味する
   - **推定A**：貞清と政定は同一人物（貞清＝政定、異名同一人）
   - **推定B**：貞清は政定の父（祖父）世代にあたり、「兵庫」の官名を継承した
   - **推定C**：貞清は信濃赤沢氏の嫡流から備中に分派した最初の人物

3. **細川氏との関係**：
   - 応永19年（1412年）に「赤澤三郎」が細川基之の奉書の使者を務めた記録あり → 室町初期から細川氏に仕えていた
   - 細川石高馬賢（1441年生）に仕えた「赤澤三郎四郎朝宗」の存在 → 「赤澤三郎」との時代差は30-40年
   - 貞清はこれらの人物の後の世代にあたり、細川野州家（鴨方細川氏）の重臣として備中に定着したと推定

**結論**：貞清の詳細は「赤沢兵庫助政定の一門と備中赤沢氏」論文（岡山県立図書館所蔵）に記載されている可能性が極めて高い。RIO自身が岡山県立図書館に問い合わせるか、デジタル岡山大百科のシステム復旧後にアクセスすることを強く推奨する。

### 69. 赤沢左兵衛門 — 四代目・西家を相続

| 項目 | 内容 |
|------|------|
| 氏名 | 赤沢左兵衛門（あかざわ さひょうえもん） |
| 世代 | 四代目 |
| 備考 | 西家を相続 |
| 父 | 宗栄（兵庫助、左兵衛）と推定 |
| 活動時期 | 慶長〜元和年間（1600年代前半）と推定 |

#### 再調査結果

**「東家・西家」の意味について**：

Web検索では「備中赤澤氏の東家・西家」に関する直接的な史料は見つからなかったが、以下の推定が可能である。

1. **分家の構造**：
   - 備中赤沢氏（柏島を本拠）が、ある時期に**東家と西家**の二家に分かれた
   - これは戦国〜江戸初期の武家にしばしば見られる本家・分家の構造
   - 柏島の「東」と「西」は地理的な分割を反映している可能性がある（柏島内の東側居住区と西側居住区）

2. **左兵衛門の位置づけ**：
   - 宗栄の通称に「左兵衛」があることから、左兵衛門は宗栄の子で通称を継承したと推定
   - 「四代目」は森本松山城主としてではなく、**柏島の在地領主（庄屋格）としての4代目**である可能性
   - 関ヶ原後に城を失い在地化した赤沢氏の中で、西家を継承した
   - 宗栄 → 左兵衛門 という直接の親子関係が最も自然

3. **関ヶ原との関連**：
   - 関ヶ原の戦い（1600年）で細川元通とともに西軍に属した備中赤沢氏は、敗戦により城と領地を失った
   - 宗連（修理進）は細川元通に従い長州に移った
   - **左兵衛門は備中に残留し、在地の庄屋・百姓として柏島に土着した一族の代表**と推定
   - 「西家」は長州に移った宗連系（東家？）との対比で、備中残留組を指す可能性がある

4. **別の可能性**：
   - 「東家」＝ 備中（東）に残った系統
   - 「西家」＝ 長州（西）に移った系統
   - 左兵衛門が「西家を相続」とは、長州に移った系統の家督を継いだ可能性
   - しかし「四代目」とあることから、在地で4代続いた系統と考える方が自然

**結論**：東家・西家の正確な意味は、備中赤澤氏サイト（akazawa-shi.jimdofree.com）に記載されている可能性が高い。サイトへのアクセスが制限されているため確認できなかったが、RIO自身でブラウザから直接アクセスすることを推奨。

### 77. 赤沢出羽守（あかざわ でわのかみ）— 板西城三人衆

| 項目 | 内容 |
|------|------|
| 氏名 | 赤沢出羽守 |
| 官位 | 出羽守 |
| 役職 | 板西城三人衆の一人 |
| 主君 | 赤沢宗伝（信濃守） |

#### 再調査結果

**新たに判明した情報**：

1. **赤沢氏と阿波小笠原氏の関係**：
   - 阿波小笠原氏は鎌倉時代に阿波守護となった信濃小笠原氏の庶流
   - 三好氏、七条氏、高志氏、赤沢氏はいずれも阿波小笠原氏の支流・同族
   - **赤沢出羽守は宗伝と同じ小笠原一族の出身であることがほぼ確実**

2. **赤沢重清との関連**：
   - No.95の赤沢重清が「出羽守」を称し、板西城主であった
   - 赤沢出羽守（三人衆）は重清の子孫か、重清本人の別名の可能性
   - ただし、重清が宗伝以前の城主であれば時代が合わない
   - **赤沢出羽守は宗伝の一族で、重清系の「出羽守」を称する分家筋**と推定

3. **中富川の戦いでの動向**：
   - 板西城三人衆として2,000の兵の指揮に関わった
   - 討死の明確な記録なし（生存の可能性あり）
   - 安芸飛騨守（三人衆の一人）は戦後に犬伏で帰農 → 出羽守も同様に帰農した可能性

**結論**：赤沢出羽守は宗伝の一族（小笠原氏出身）で、板西城の軍事面の中核を担った重臣。中富川の戦いでの生死は不明だが、三人衆の安芸飛騨守が生存・帰農していることから、出羽守も同様に生き延びた可能性がある。

### 1. 備中赤沢氏の「関ヶ原後→大坂の陣」伝承

備中赤澤氏サイト管理人（zawa氏）の記述に基づく伝承：

> 「赤澤一族は関ヶ原で改易となり、それ以前の豊臣家の恩顧に報いるために備中より大坂に入城し、真田隊に一族上げて所属し多くが討ち死にした」

**この伝承の歴史的検証**：

1. **時系列の整合性**：
   - 慶長5年（1600年）関ヶ原 → 細川元通とともに西軍で敗北
   - 元和元年（1615年）大坂夏の陣 → 15年の空白期間
   - 関ヶ原で改易された浪人が大坂の陣で豊臣方に参じるのは歴史的に多数の事例あり（真田信繁、後藤又兵衛、長宗我部盛親など）
   - **時系列は矛盾しない**

2. **豊臣家の恩顧**：
   - 備中赤沢氏は細川氏（鴨方細川氏）の家臣として、間接的に豊臣政権下で活動
   - 文禄元年（1592年）の朝鮮出兵にも参加しており、豊臣家との繋がりがある
   - 関ヶ原後に改易され浪人化した一族が、大坂の陣で豊臣方に参じる動機は十分

3. **真田隊への所属**：
   - 大坂の陣には約10万の豊臣方が参集。うち7〜8万は浪人
   - 真田信繁（幸村）の部隊には多数の浪人が所属
   - 備中からの浪人が真田隊に所属することは地理的にも可能
   - **ただし、真田隊の具体的な構成員名簿に「赤沢」の名は確認できなかった**

4. **討死の規模**：
   - 「多くが討ち死にした」 → 一族の大半が戦死した
   - 大坂夏の陣で真田隊は壊滅的な損害を受けており、所属する浪人の多くが戦死するのは歴史的事実と整合
   - **この伝承が事実であれば、備中赤沢氏の主要な男性構成員の大半がこの戦いで失われたことになる**

**結論**：この伝承は歴史的にあり得る内容であり、時系列にも矛盾がない。ただし、裏付ける一次史料は未発見。備中赤澤氏サイトの管理人zawa氏が根拠とした史料（家伝・過去帳等）の確認が必要。
