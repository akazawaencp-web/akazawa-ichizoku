## 安土桃山〜江戸初期の赤沢氏

### 61. 貞経（小笠原貞経）

**基本情報**

| 項目 | 内容 |
|------|------|
| 氏名 | 赤沢経直 → 小笠原貞経（おがさわら さだつね） |
| 生没年 | 天文12年（1543年）〜 慶長19年3月19日（1614年4月27日） |
| 享年 | 72歳 |
| 官位 | 伊豆守 |
| 身分 | 徳川幕府旗本（小笠原平兵衛家の祖） |
| 父 | 赤沢経智（三男） |
| 兄 | 赤沢長勝 |
| 妻 | 相馬氏家臣・黒木修理進の娘 |
| 子 | 経治、忠経、豊経、守経（四男） |

**生涯と事績**

1. **信濃時代**：父・経智とともに小笠原長時に仕えたが、武田信玄の侵攻により信濃国から追放される
2. **上洛と兄の戦死**：経智とともに三好長慶を頼り上洛。永禄元年（1558年）、兄・長勝が北白川の戦いにおいて討死
3. **奥州相馬氏に身を寄せる**：兄の死後、陸奥国中村の相馬盛胤・義胤父子に招聘される
4. **軍功**：相馬氏が伊達氏の坂元城を攻めた際に一番乗りの軍功を挙げ、鎧作原の戦いでも首級を上げる

**小笠原流弓馬術礼法の継承**

- **永禄5年（1562年）**：小笠原家の総領・小笠原長時から「糾法的伝」（弓馬術礼法の奥義）と系図・記録を託され、弓馬術礼法の宗家の道統を継承
- **元亀3年（1572年）6月2日**：長時から正式に小笠原流弓馬術礼法の道統を託される
- **天正3年（1575年）8月12日**：師範の許状を受ける
  - ※一説では、小笠原貞慶が武田信玄に敗れた際に、弓馬礼法の伝統を絶やさないため赤沢経直に三法を総取仕切る宗家の道統を託したとも

**徳川家康への仕官と復姓**

- **慶長9年（1604年）**：徳川家康に拝謁
- その後、徳川秀忠に仕える
- **廩米500俵**を賜る
- 赤沢から**小笠原に復姓**
- 旗本小笠原平兵衛家の祖となる

**復姓の理由**（※推定）：
- 小笠原流弓馬術礼法の正統な継承者であり、その道統を明確にするため小笠原姓に戻したものと考えられる
- 徳川幕府における弓馬術礼法師範としての権威付けの意味合いもあったと推測される

**平兵衛家のその後**

- 歴代将軍に仕え、将軍家子女の婚礼や元服の儀式に携わる
- 8代将軍・徳川吉宗の命により復興された流鏑馬（騎射挟物）が第20代小笠原常春に預けられる
- 以後、代々騎射師範として門弟を指揮
- 御維新（明治維新）まで高家として幕府の弓馬術礼法の師範を務める
- 貞経の子・経治の三男に小笠原直経（元和4年/1618年生）がおり、寛永13年（1636年）に3代将軍・徳川家光に拝謁
- 現代では小笠原流（弓馬術礼法小笠原教場）宗家として存続

**小笠原流の二家体制**

徳川幕府における小笠原流は以下の二家が伝承した：
1. **小笠原縫殿助家**：京都小笠原氏庶流の子孫
2. **小笠原平兵衛家**：赤沢氏の子孫で小笠原に復姓した貞経の家系

## 備中・長州系赤沢氏

### 備中赤沢氏の背景

備中赤沢氏は、清和源氏小笠原氏の庶流である赤沢氏の一族が備中国に展開したものである。備中国浅口郡は細川氏（鴨方細川氏・野州家）の支配下にあり、赤沢氏はその家臣として活動した。

**鴨山城と細川氏**
- 応永14年（1407年）、細川満国が備中浅口郡に鴨山城を築城
- 鴨方細川氏は管領細川家の後裔で、浅口郡・矢田郷、伊予宇摩郡・温泉郡を領有
- 七代鴨山城主・細川通董（通薫）が天正3年（1575年）に鴨山城を奪還
- 天正15年（1587年）、通董は九州征伐に際し赤間関（下関）にて没
- 通董の嫡男・細川元通が家督を継承

**関ヶ原後の展開**
- 慶長5年（1600年）関ヶ原の戦いで西軍に属した毛利氏が周防・長門の2ヶ国に減封
- 細川元通は鴨山城を失い、義兄弟の毛利秀元のもとに身を寄せる
- 元通の子孫は長府毛利家の家老となり、明治維新を迎える
- 赤沢氏の一部もこれに従い長州に移動した

| 通称 | 兵庫助 |
| 官位 | 兵庫頭 |
| 生没年 | 情報なし |

**調査結果**

- 岡山県立図書館（デジタル岡山大百科）に「**赤沢兵庫助政定の一門と備中赤沢氏―細川典厩家・野州家との関わりを中心に―**」という学術論文が所蔵されている
- この論文は細川典厩家（管領家の家老格）と野州家（細川氏の分家）と備中赤沢氏の関係を中心に論じている
- 赤沢兵庫助政定は、細川典厩家・野州家の重臣として備中に勢力を持った人物と推定される
- ※延徳元年（1489年）、細川政元が主催した犬追物に「赤沢兵庫助」が参加した記録がある。この赤沢兵庫助が政定本人か、同族の者かは不明
- ※備中赤沢氏では「兵庫」の官職名が継承されていることから、政定は備中赤沢氏の重要人物であり、宗春や貞清との関係も想定される

**柏島神社との関連**：
- 天正12年（1584年）3月、「赤沢兵庫頭政信」が柏島（現・倉敷市玉島柏島）に八幡宮（現・柏島神社）を創建した記録がある
- この「赤沢兵庫頭政信」は政定の子孫か同族の者と推定される

- ※備中浅口郡の応仁期（1467〜1477年）には「赤澤修理亮」が鴨山城の重臣として森本松山城に在城したとの記録があり、「修理」の官職名が赤沢氏の中で代々使われていた可能性がある
- 赤沢氏の長州展開の端緒となった人物

**長州への移動の背景**：
- 関ヶ原の戦い（1600年）後、毛利氏は広大な領地を失い周防・長門の2ヶ国に減封された
- 細川元通は備中の鴨山城を失い、毛利秀元のもとに身を寄せた
- 赤沢宗連はこれに従い、備中から長州に移住した
- 長州での赤沢氏の展開については、長州藩の家臣団名簿等に記録がある可能性があるが、Web上では確認できなかった

## 備中赤沢氏の全体像

### 系譜の推定

```
貞清（赤沢太郎、兵庫頭）
  └─ 宗春（左馬助、兵庫卿）【森本松山城主・初代】
       ├─ 宗照（五郎四郎、兵庫助）【森本松山城主・2代目】〜1570年頃討死
       │    ├─ 宗栄（兵庫助、左兵衛）【森本松山城主・3代目】
       │    │    └─ 左兵衛門【4代目、西家を相続】
       │    ├─ 宗台（久助宗白？）〜1600年 関ヶ原で討死
       │    ├─ 僧（海徳寺）
       │    └─ 宗連（修理進）→ 長州に移る
       ├─ 福寿院
       ├─ 小右衛門
       └─ 九兵衛

※政定（兵庫頭）は貞清と同世代か近縁の別系統の可能性あり
```

### 森本松山城（「山の城」）について

| 項目 | 内容 |
|------|------|
| 名称 | 森本松山城 |
| 所在地 | 備中国浅口郡（現・岡山県倉敷市玉島柏島） |
| 性格 | 鴨山城（鴨方細川氏の本城）の支城・出城 |
| 城主 | 赤沢宗春 → 宗照 → 宗栄（3代） |
| 機能 | 浅口郡南部の海上交通（玉島港方面）の監視・防衛拠点（※推定） |

- 「備中浅口郡山の城」は森本松山城を指すと考えられる
- 備中松山城（高梁市）とは全くの別城である
- 鴨山城の支城として、玉島港・柏島周辺の海上交通を押さえる要衝に位置していた

### 細川氏との関係

| 細川氏の当主 | 時期 | 赤沢氏との関係 |
|-------------|------|-------------|
| 細川満国 | 応永14年（1407年）〜 | 鴨山城築城、浅口郡領有 |
| 細川通政 | 戦国中期 | 宗照が仕える、森本松山城主に任じる |
| 細川通董（通薫） | 天正3年（1575年）〜天正15年（1587年） | 宗照・宗栄兄弟が仕える |
| 細川元通 | 天正15年（1587年）〜 | 宗栄・宗連が仕える、朝鮮出兵に同行 |

### 関ヶ原の戦いと備中赤沢氏

1. **西軍に属す**：細川元通は毛利氏に従い西軍に属した。赤沢氏も細川元通の家臣として参戦
2. **宗台（久助宗白？）の討死**：近江国膳所の赤沢谷にて敵軍に囲まれ討死
3. **鴨山城の喪失**：毛利氏の減封に伴い、細川元通は鴨山城を失う
4. **一族の分散**：
   - 宗連（修理進）は細川元通に従い長州に移る
   - 一部は備中に残留（在地の庄屋・百姓として存続か）
   - 一部は長州藩の家臣として防長に移住

### 赤沢氏と柏島

- 柏島（倉敷市玉島柏島）は備中浅口郡に属する地で、森本松山城の所在地
- 天正12年（1584年）、赤沢兵庫頭政信が柏島に八幡宮（現・柏島神社）を創建
- 宗栄の居所が「柏島の山崎」とされていることから、柏島は赤沢氏の本拠地であった
- 現在も倉敷市玉島地区に赤沢（赤澤）姓の住民がいる可能性がある

### 朝鮮出兵（文禄・慶長の役）との関係

- 文禄元年（1592年）、宗栄と久助宗白（宗台？）が細川元通に従い朝鮮に渡海
- 蔚山などの戦いに参加
- この朝鮮出兵への参加は、備中赤沢氏が細川氏の有力家臣であったことを示す

## 調査上の注意事項

1. **備中赤澤氏の専門サイト**（https://akazawa-shi.jimdofree.com/）にアクセスが制限されており、詳細情報の取得ができなかった。このサイトには宗春〜宗連の詳細な系図・事績が掲載されている可能性が高い。RIO自身でアクセスして確認することを強く推奨する。

2. **岡山県立図書館の論文**「赤沢兵庫助政定の一門と備中赤沢氏」は、政定・宗春・宗照・宗栄ら備中赤沢氏の詳細な研究。岡山県立図書館で閲覧するか、デジタル岡山大百科で確認できる可能性がある。

3. **朝治（No.62）、式部少輔（No.63）、左兵衛門（No.69）、僧（No.71）、福寿院（No.73）、小右衛門（No.74）、九兵衛（No.75）** については、一般公開のWeb資料では詳細情報が見つからなかった。これらの人物の情報は、赤沢家の私的な系図資料や地方史料にのみ記録されている可能性が高い。

4. **「宗台」と「久助宗白」の同一性**は確認できていない。両者が同一人物か別人かは、系図の精査が必要。

5. **備中松山城（高梁市）と森本松山城（倉敷市玉島柏島）の混同に注意**。赤沢氏が城主であったのは後者。
