### 主な事績・経歴

- **寛治元年（1087年）**: 父・義光が後三年の役に無断参戦し、朝廷から官職を解任される。以後、義光は常陸国での勢力基盤構築に注力。
- **嘉承元年（1106年）**: 父・義光と共に従兄・源義国と関東で合戦。
- **天仁2年（1109年）**: 義兄（母の兄弟）の鹿島三郎成幹が、源氏の棟梁・源義忠を暗殺する大事件が発生。この事件は義光一族にも大きな影響を与えた。
- **長承2年（1133年）**: 病没。

### 人物像・歴史的意義

義業は自身は「佐竹」姓を名乗っていないが、実質的な佐竹氏の祖と位置づけられる。常陸国で勢力を築くことができた最大の要因は、**母と妻がともに常陸の有力豪族・吉田清幹（常陸大掾）の娘**であったことにある。この婚姻戦略により、地元豪族との強固な結びつきを得て、常陸国久慈郡佐竹郷に根を下ろした。

義業の子・昌義が佐竹郷を本拠とし、正式に佐竹氏を称した。佐竹氏はその後、鎌倉〜戦国〜江戸時代を通じて常陸の有力大名として存続し、秋田藩主として明治まで続く名門となる。

### 小笠原氏・赤沢氏との関係性

義業の弟・義清が甲斐源氏の祖であり、そこから武田氏・加賀美氏・小笠原氏・赤沢氏が生まれた。義業は赤沢氏から見れば、甲斐源氏の祖・義清の兄にあたる。また、加賀美遠光の母は「佐竹義業の末娘」とする系図もあり、直接的な血縁関係も存在する。

### 墓所・ゆかりの地

- 常陸国久慈郡佐竹郷（現・茨城県常陸太田市周辺）が本拠地。
- 墓所の詳細な所在地は情報なし。

### 後世の評価

佐竹氏の実質的な始祖として、茨城県の歴史において重要な位置を占める。父・義光の策略的な性格を受け継ぎつつ、婚姻政策を通じて常陸に根を下ろした政治力が評価される。

### 主な事績・戦歴

- **保延6年（1140年）**: 13歳で武田八幡宮にて元服し「武田太郎信義」と名乗る。
- **治承4年（1180年）**: 以仁王の令旨を受けて甲斐で挙兵。源頼朝とは別個に独立した軍事行動を展開。
  - 波志田山で平家方の大庭軍と戦闘。
  - 9月に信濃国諏訪に攻め込み勢力を拡大。
  - **10月・富士川の戦い**: 甲斐源氏の主力として平維盛軍を大敗させる。この戦いでは水鳥の羽音で平家軍が自壊したという逸話が有名だが、実際には信義らの夜襲が決定打だったとされる。
- 富士川の戦いの後、**駿河守護**に任じられる（頼朝による任命とされるが、実態は信義自身の実力で駿河を制圧）。
- **元暦元年（1184年）**: 一ノ谷の戦い、壇ノ浦の戦いなどに参加。
- **元暦元年（1184年）6月**: 嫡男・一条忠頼が頼朝の命により、鎌倉の宴席で謀殺される。
- **文治2年（1186年）3月9日**: 病没。

### 人物像

源頼朝と並ぶ「もう一人の源氏の棟梁」とも呼ぶべき存在。甲斐源氏は頼朝とは独立した軍事勢力であり、信義は当初、頼朝と対等の立場で行動していた。しかし頼朝の政治的手腕により次第に従属を強いられ、嫡男を殺され、失意のうちに没した。

大河ドラマ『鎌倉殿の13人』では八嶋智人が演じ、「我こそは源氏の棟梁」と自負する誇り高い武将として描かれた。

### 小笠原氏・赤沢氏との関係性

信義は源清光の次男であり、加賀美遠光の兄（もしくは甥 ※系図により異同あり）にあたる。武田氏と小笠原氏は同じ甲斐源氏の一族であり、後に信濃をめぐって複雑な関係となる。赤沢氏から見れば、信義は大叔父（もしくは従祖父）の世代にあたる。

### 墓所・ゆかりの地

- **願成寺**（山梨県韮崎市神山町鍋山）: 武田信義の墓所。五輪塔は鎌倉様式。
- **武田八幡宮**（山梨県韮崎市）: 元服の地。
- **武田信義館跡**（韮崎市周辺）。

### 後世の評価

武田信玄の直系の祖として、甲斐の歴史において極めて重要な人物。源平合戦における甲斐源氏の軍事力の中核であり、富士川の戦いの実質的な勝利者。しかし頼朝との政治的駆け引きに敗れ、嫡男を失った悲劇の武将でもある。

### 主な事績・経歴

- **承安元年（1171年）**: 宮中で怪異が発生。高倉天皇が「源氏の弓矢の名手」として遠光を召し、**鳴弦の術**（弓弦を鳴らして邪気を祓う儀式）を行わせたところ怪異が治まった。褒賞として**不動明王像**と**近江国志賀郡**を下賜された。当時、遠光は滝口武者であったとされる。
- **治承4年（1180年）〜**: 治承・寿永の乱に次男の小笠原長清と共に参加。
- **文治元年（1185年）**: 平家滅亡後、源頼朝より「**御門葉**」（将軍一門格）の一人として特別な待遇を受け、**信濃守**に任じられる。
- **晩年**: 真言宗の古刹・法善寺を再興。遠光寺（現・日蓮宗）を創建。

### 人物像

弓矢の名手として朝廷にも名が知られた武人でありながら、政治的にも巧みに立ち回った。頼朝の御門葉に列せられたことは、甲斐源氏の中でも特別な信任を得ていたことを示す。ただし、長男の秋山光朝は頼朝に疎まれて謀殺されており、甲斐源氏に対する頼朝の猜疑心は遠光の一族にも及んだ。

88歳の長命は、この時代としては極めて異例。

### 小笠原氏・赤沢氏との関係性

**赤沢氏の直系の祖父の父**にあたる。遠光→長清→長経→清経という系譜で、赤沢氏は遠光の曾孫から始まった。小笠原氏・南部氏・秋山氏の共通の祖であり、これらの氏族を生んだ点で、日本の武家史上きわめて重要な人物。

### 墓所・ゆかりの地

- **遠光寺**（山梨県甲府市伊勢）: 遠光が建暦元年（1211年）に自ら建立した菩提寺。現在は日蓮宗。
- **法善寺**（山梨県南アルプス市加賀美3509）: 遠光の館跡に孫の遠光経が建立した寺。
- **加賀美遠光館跡**（山梨県南アルプス市）: 市指定史跡。
- **加賀美遠光廟所**（山梨県南アルプス市小笠原376）。
- **熊野神社**（山梨県南アルプス市秋山）にも墓所あり。

### 後世の評価

小笠原氏・南部氏という二大武家の祖として、その歴史的影響力は計り知れない。特に南部氏は奥州に大きな勢力を築き、戦国時代〜江戸時代を通じて盛岡藩・八戸藩として存続した。鳴弦の術による宮中での活躍は、武芸に秀でた甲斐源氏の象徴的なエピソードである。

- **治承4年（1180年）**: 以仁王の令旨を受けて父・遠光と共に源平合戦に参加。当時19歳。
- **養和元年（1181年）2月1日**: 頼朝の仲介で有力御家人・上総広常の娘を妻に迎える。
- **文治5年（1189年）**: 奥州合戦に参加。
- **建久4年（1193年）3月21日**: 鶴岡八幡宮の流鏑馬で、22人の射手の一人として選ばれ射手を務める。
- **承久3年（1221年）**: **承久の乱**で東山道方面の大将軍を務め、子息8人と共に京都へ攻め上る。大きな軍功を挙げ、乱後に「**七ヶ国管領**」となる。
- **阿波国守護**: 承久の乱の恩賞として阿波国守護に任じられる。以後、鎌倉時代を通じて小笠原氏は阿波守護職を保持。
- **仁治3年（1242年）7月15日**: 信濃にて81歳で死去。

### 弓馬術礼法

**「弓馬四天王」**の一人として称えられた。他の三名は武田信光・海野幸氏・望月重隆。

『吾妻鏡』には「弓馬の術に優れ」と記録されており、鎌倉幕府の流鏑馬・騎射の第一人者であった。長清が体系化した弓馬術の知識・技術は、後世「**小笠原流**」として武家の礼法・弓術の基本となり、現在も**弓馬術礼法小笠原教場**として継承されている。

### 人物像

81歳の長命が示すように、逆境にも粘り強く対応できる人物だったと推測される。承久の乱では60歳近い高齢でありながら東山道軍の大将として出陣し、子息8人を率いて戦功を挙げた。弓馬に秀でた武人であると同時に、礼法を重んじた文武両道の人物。

### 小笠原氏・赤沢氏との関係性

小笠原氏の初代当主であり、**赤沢氏の祖・清経の祖父**にあたる。長清→長経→清経という系譜。長清が築いた信濃・阿波の所領基盤が、子孫の赤沢氏にも受け継がれた。

### 墓所・ゆかりの地

- **小笠原長清供養塔**（山梨県北杜市明野町上手2409）。
- **小笠原長清公館跡**（山梨県南アルプス市）: 市指定史跡。
- 高野山奥の院にも供養塔がある。

### 後世の評価

武家の礼法の創始者として、日本文化史上にも大きな足跡を残した。「小笠原流」は弓術・馬術・礼法を総合した武家の教養体系として江戸時代に大成し、現代にも続いている。また、信濃守護・阿波守護として広域に影響力を持った戦略家でもあった。

### 主な事績・戦歴

- **文治5年（1189年）**: 父・長清に従い**奥州合戦**に参加（10歳）。
- **正治元年（1199年）**: 二代将軍・**源頼家の近習**として出仕。頼家が十三人の合議制に反発した際、目通り（直接の面会）が許される**5人の近習**の一人に選ばれる。流鏑馬の射手も務めた。
- **建仁3年（1203年）**: **比企能員の変**で比企勢力とみなされ拘禁される。その後鎌倉を離れる。
- **承久3年（1221年）**: **承久の乱**で父・長清とともに東山道軍として出陣し、京へ攻め上る。大きな軍功を挙げる。
- **貞応2年（1223年）**: 父より家督を継承し、**阿波国守護**となる。
- **宝治元年（1247年）5月9日**: 京都の新日吉社で流鏑馬の神事を務める（出家後「小笠原入道」と称す）。
- **晩年**: 信濃国**伊賀良荘松尾**（現・長野県飯田市）に隠棲。

### 人物像

源頼家の側近として若くして政治の中枢に関わり、流鏑馬の名手として武芸にも秀でた。比企の変で一時的に失脚するが、承久の乱で復帰し、阿波守護の地位を確保した。晩年は信濃に帰り、伊賀良荘松尾で隠棲した。

### 小笠原氏・赤沢氏との関係性

長経は**赤沢清経の父**であり、赤沢氏が生まれた直接の起点。次男の清経が伊豆国赤沢郷を領して赤沢氏を称した。長男の長忠は松尾小笠原氏の祖となり、長経の子孫は信濃各地に分かれて小笠原一族の枝族を形成した。

### 墓所・ゆかりの地

- **信濃国伊賀良荘松尾**（長野県飯田市）: 晩年の隠棲地。
- 詳細な墓所の所在は情報なし。

### 後世の評価

小笠原氏第2代として、父・長清の遺産を継承し信濃・阿波の支配を確立した。源頼家の近習という経歴は、小笠原氏が鎌倉幕府の中枢に近い存在であったことを示す。今上天皇の直系祖先の一人としても系図上確認されている。

## 21. 赤沢清経（あかざわ きよつね）— 赤沢氏初代

### 基本情報

| 項目 | 内容 |
|------|------|
| **生没年** | 情報なし（鎌倉時代前期、13世紀前半に活動と推定） |
| **別名** | 源二郎、六波羅二郎 |
| **官位** | 伊豆守、山城守（※系図により異同あり） |
| **氏族** | 甲斐源氏・小笠原氏庶流・赤沢氏初代 |
| **本貫** | 伊豆国田方郡赤沢郷（現・静岡県伊東市赤沢） |

### 家族関係

| 続柄 | 人物 |
|------|------|
| **父** | 小笠原長経（小笠原氏第2代） |
| **母** | 本三位中将・平重衡の娘（※系図の記載。重衡は平清盛の五男で南都焼討の実行者として知られる） |
| **兄** | 小笠原長忠（松尾長忠） |
| **子** | 長忠（赤沢長忠、大蔵少輔） |

### 赤沢氏創設の経緯

清経は小笠原長経の次男として生まれ、**伊豆国田方郡赤沢郷**（現在の静岡県伊東市赤沢）を所領として与えられた。この地名を氏の名として「赤沢」を称し、赤沢氏の祖となった。

初代清経が**伊豆守**に任ぜられたことから、以後の赤沢氏当主も伊豆守を称することが多かった。

「六波羅二郎」の通称は、京都の六波羅に居住していた時期があったことを示唆する。鎌倉幕府の六波羅探題に関連する任務に就いていた可能性がある。

その後、清経は伊豆から**信濃国埴科郡埴生**（現・長野県千曲市）に移住した。以降、赤沢氏は主に信濃を本拠とする。

### 母の系譜 — 平重衡の娘

系図に記載された清経の母「本三位中将・平重衡の娘」が事実であれば、赤沢氏は源氏の血統のみならず平氏（平清盛の孫）の血も引くことになる。ただし、平重衡は元暦2年（1185年）に処刑されており、清経の生年との整合性については議論がある。※推定として、重衡の遺児が小笠原家に嫁いだ可能性は否定できない。

### 赤沢の地名の由来

**伊豆国田方郡赤沢郷**（現・静岡県伊東市赤沢）は、伊豆半島東岸に位置する地域である。

「赤沢」という地名の由来について、確定的な記録は見つからなかったが、以下の推定がある:

1. **赤土説**: 伊豆半島は火山活動が活発な地域であり、赤褐色の火山性土壌（赤土）が露出した沢（谷）があったことから「赤い沢」→「赤沢」と名付けられた。※推定
2. **鉄分含有説**: 鉄分を多く含む湧水が赤褐色を帯びていた沢があった。赤沢温泉の存在は地下の鉱物質の豊富さを示唆する。※推定
3. **字義説**: 「赤」は「明るい」「開けた」の意味もあり、明るく開けた沢の意とする説。※推定

現在の伊東市赤沢は、伊豆高原に隣接する太平洋を望む自然豊かな地域で、赤沢温泉郷（約24万平方メートルの広大なリゾートエリア）として知られる。

### 歴史的意義

赤沢氏は、小笠原氏の正統な庶流として成立した。清和源氏→甲斐源氏→小笠原氏→赤沢氏という流れは、日本の武家社会における名門中の名門の系譜である。清経が伊豆から信濃に移った経緯は、小笠原氏の信濃支配の一環として位置づけられる。

### 墓所・ゆかりの地

- **伊豆国田方郡赤沢郷**（静岡県伊東市赤沢）: 赤沢氏発祥の地。
- **信濃国埴科郡埴生**（長野県千曲市）: 清経が移住した地。
- 具体的な墓所の情報はなし。

鎌倉時代末期〜南北朝時代の動乱期に向けて、赤沢氏が軍事的な拠点を築いていたことが確認できる。
