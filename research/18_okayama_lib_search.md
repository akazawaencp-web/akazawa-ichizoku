## 6. Web調査で判明した重要情報

### 6-1. 「備中赤澤家と玉島古戦記」サイト

- **URL**: https://akazawa-shi.jimdofree.com/
- **内容**: 備中赤澤氏の歴史を専門的にまとめた個人サイト
- **サブページ**:

#### サイトから得られた備中赤澤氏の系譜情報

**初代: 赤澤兵庫助宗照（むねてる/むねあき）**
- 野洲家（細川氏の一族）当主・細川通政に仕える
- 細川通政は鴨山城（現・浅口市鴨方町）に拠る
- 森本松山城（現・倉敷市玉島柏島）の城主となる
  - 森本松山城は鴨山城の支城
- 父・宗春は主君・細川通政に従い勝原原の戦いで討死
- 尼子氏の圧力により細川通政が浅口を棄てた後も、宗照は玉島に残留
- 元亀元年（1570年）、尼子残党・秋上綱平の備中侵入に際し、毛利方として柏島で戦い討死

**二代: 赤澤宗栄（そうえい）**
- 宗照の子
- 細川通政の子・細川通薫に仕える

**三代: 赤澤久助宗伯（きゅうすけ そうはく）**
- 宗栄の子
- 文禄元年（1592年）、細川通薫に従い朝鮮半島に渡海（蔚山等で戦闘）
- 慶長5年（1600年）、関ヶ原の戦いで西軍に属す
- 近江国善正寺（膳所）で奮戦するも討死
  - 討死の地は「赤沢谷」と呼ばれたとの伝承あり

**その後**: 関ヶ原後、旧領に残った赤澤一族の多くは帰農し、歴史の表舞台から姿を消した。

#### 森本松山城に関する追加情報
- 所在地: 倉敷市玉島柏島
- 南北朝期: 藤田小次郎が居城
- 応仁期: 鴨方細川氏の家臣・赤沢修理介が居城
- 天文7年（1538年）11月15日: 「柏島清所」（森本松山城のこと）で合戦が発生（同年11月26日付文書で確認）
- 源平合戦の水島古戦場記念碑が城跡の一角にある

#### 注目点
- サイトでは「赤沢兵庫助宗照」としており、「政定」ではなく「宗照」の名前を使用
- 延徳元年（1489年）の犬追物に「赤沢兵庫助」が参加（細川政元主催）との記録あり
- 細川典厩家の被官人に「赤沢兵庫助政真」の名が見える
- **→ 「赤沢兵庫助政定」は「宗照」「政真」等と同一人物、あるいは混同の可能性あり**

### 6-2. 赤沢氏の全体系譜（小笠原諸流系図）

『小笠原諸流系図』による赤沢氏本流の系譜:
```
長経 → 清経（号赤沢源次郎）→ 安経 → 経顕 → 氏経 → 常興 → 経光 → 武経 → 満経 → 経隆 → 朝経 → 政経 → 経智 → 貞経
```

- **初代・清経**: 小笠原長清の嫡男長経の次男。伊豆国田方郡赤沢郷を本貫として赤沢を称す。後に信濃国埴科郡埴生に移住。
- **朝経（宗益）**: 1491年頃から管領・細川政元に仕え、内衆（側近）として数カ国の守護代を務める。1507年、永正の錯乱で敗死。

### 6-3. 赤沢朝経（宗益）の事績
- 信濃国塩崎城が本拠
- 小笠原流弓術を細川政元に伝授し、接近
- 1499年: 細川政元の命により延暦寺を攻撃
- 大和国・山城国等で軍事活動
- 1507年: 細川政元暗殺（永正の錯乱）に際し、丹後で敗死
- 嫡子・政経が家督を継承するも1509年に信濃に帰還
- 弟・長経は阿波細川氏（細川澄元）に仕える

### 6-4. 赤沢宗伝（阿波赤沢氏）
- 板西城（徳島県板野郡板野町）城主
- 三好氏に仕え、3000貫の領地を保有
- 「板西城三人衆」「赤沢家十二人衆」を統制
- 1582年、中富川の戦いで長宗我部元親に敗れ討死
- 末裔は讃岐（香川県東かがわ市）に移り、勝覚寺を開基

### 6-5. 鴨山城（浅口市鴨方町）
- 1407年（応永14年）、細川満国が築城
- 細川満国は備中・浅口郡・山田郷等を領有
- 備中野洲家（細川氏分家）の居城として機能
- 赤沢氏はこの鴨山城の支城・森本松山城を任された

### 6-6. 備中細川氏（野洲家）の系譜
```
細川満国 → ... → 細川通政（鴨山城）→ 細川通薫
```
- 細川通政: 足利義輝から「輝」の字を拝領し「輝政」とも称す（架空説あり）
- 赤沢兵庫助宗照は通政に仕えた

## 10. 調査の総括

### 確定した事実
- 赤沢氏は小笠原氏の庶流で、信濃国を本拠とした武家
- 赤沢朝経（宗益）が細川政元の内衆として活動（1491年頃～1507年）
- 備中赤澤氏の実質的初代は赤澤兵庫助宗照（細川通政の家臣、森本松山城主）
- 森本松山城は鴨山城（浅口市鴨方町）の支城で、現在の倉敷市玉島柏島に所在
- 赤澤氏は備中野洲家（細川氏分家）に仕え、毛利氏の支配下でも活動
- 関ヶ原の戦い（1600年）後、備中赤澤氏は衰退・帰農

### 未確定の論点
- 「赤沢兵庫助政定」と「赤沢兵庫助宗照」「赤沢兵庫助政真」の関係（同一人物か別人か）
- 「赤沢兵庫助政定の一門と備中赤沢氏」論文の著者・掲載誌（特定できず）
- 備中赤沢氏と信濃赤沢氏本流の正確な系譜的繋がり
