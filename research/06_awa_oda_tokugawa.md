### ■ 板西城（ばんざいじょう）の概要

| 項目 | 内容 |

| **規模** | 板野郡一帯を支配。赤沢家は3,000貫の大勢力 |

#### 板西城の歴史的経緯
1. **平安末期**: 近藤六親家が築城
2. **鎌倉時代**: 親家に子がなく、**赤沢兵庫頭宗定**を養子に迎えて城を譲る。親家は新居見城へ移転
3. **室町〜戦国初期**: 宗定にも子がなく、小笠原氏一族が赤沢家を継承。以後、赤沢氏が板西城主を世襲
4. **戦国時代**: 三好実休に仕える小笠原宗伝が入城し、断絶していた赤沢姓を名乗って「赤沢信濃守宗伝」と称す
5. **天正10年（1582年）**: 中富川の戦いで落城。城主・赤沢宗伝以下、多くの将兵が討死

### ■ 中富川の戦い（天正10年/1582年）の詳細

| 項目 | 内容 |
|------|------|
| **日時** | 天正10年（1582年）8月28日〜9月21日（約20日間の攻防） |
| **場所** | 阿波国中富川河畔〜勝瑞城周辺（現・徳島県藍住町） |
| **攻撃側** | 長宗我部元親軍：約23,000人 |
| **防御側** | 十河存保（三好存保）軍：約5,000人 |
| **結果** | 長宗我部軍の大勝。三好方は名のある武将54人、兵800余が戦死 |
| **長宗我部側損害** | 約600人戦死 |
| **戦後** | 十河存保は勝瑞城に約1ヶ月籠城後、讃岐へ撤退。阿波国は長宗我部氏の支配下に |

#### 戦いの経過
1. 本能寺の変（1582年6月）で織田信長が死去。信長の四国征伐計画が中止
2. 長宗我部元親は好機と見て阿波への大侵攻を開始
3. 8月28日、中富川を挟んで両軍が激突。長宗我部軍は上流を堰き止めておき、十河勢が渡河を始めたところで放流する戦術を使用
4. 板西城の赤沢宗伝は約2,000の兵を率いて奮戦
5. 三好方は壊滅的敗北。城主級の武将の大半が戦死

#### 赤沢宗伝の最期
伝承によると、宗伝は敵将と組み合って奮戦していたが、**草鞋（わらじ）の紐が切れた**ところを逆転され、討ち取られたという。この伝承から、愛染院の宗伝の廟には「足腰の病に霊験がある」という信仰が生まれた。

#### 事績
- 元は小笠原氏の一族で「小笠原宗伝」と称していた。三好実休に仕え、阿波板西城に入城
- 板西城に入ると、断絶していた同じ小笠原一族である赤沢姓に改名し「赤沢信濃守宗伝」と称した
- 三好実休の姪を室に迎え、三好家と姻戚関係を結ぶ
- 配下に「板西城三人衆」（赤沢出羽守・坂上備前守・安芸飛騨守）と「赤沢家十二人衆」を組織
- 永禄5年（1562年）、久米田の戦いで主君・三好実休が討死。宗伝は上桜城主・篠原長房、木津城主・篠原自遁と共に出家
- 出家後も三好実休の子・三好長治の重臣として補佐を続ける
- 天正元年（1573年）、上桜城の戦いで篠原長房が自刃した際、宗伝も攻撃を受けたが、高野山に3年間籠り、その後板西城に復帰
- 天正10年（1582年）9月、中富川の戦いにおいて約2,000の兵を率いて三好方として奮戦するが、一族郎党と共に討死

#### 死後
- 四国八十八箇所霊場・第三番札所金泉寺の奥の院「愛染院」（板野町）に廟が祀られている
- 一子・正本は大坂天満の興正寺で得度し、天正10年（1582年）に讃岐国入野郷小砂村に「小砂坊」を開基。これが後の**勝覚寺**（香川県東かがわ市三本松。真宗興正派・海暁山海暁閣勝覚寺）となる

#### 歴史的意義
阿波における赤沢氏の最盛期を築いた人物。板西城を拠点に3,000貫の大勢力を保持し、三好家の重臣として阿波の政治・軍事に大きな影響力を持った。中富川の戦いでの壮絶な討死は、阿波赤沢氏の終焉を象徴する。

## 第二部：織田〜豊臣〜徳川期の赤沢氏

### ■ 時代背景

天正10年（1582年）の本能寺の変とその後の動乱期において、信濃の赤沢氏は大きな転換点を迎えた。

- **天正10年（1582年）**: 武田氏滅亡。織田信長が信濃を制圧。信長の死後、信濃は混乱状態に
- **天正11年（1583年）**: 小笠原貞慶が徳川家康の支援を得て深志城（松本城）を回復。この際、武田時代に離反した赤沢氏・塔原氏などを粛清
