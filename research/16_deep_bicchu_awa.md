### 1. 鴨山城と細川野州家の全容

#### 城の構造

鴨山城は南北に細長い尾根を利用した山城で、全長約300m、平均幅20m。北端から南へ「一ノ壇」「二ノ壇」「三ノ壇」「四ノ壇」「五ノ壇」が連なり、さらに「陽の郭」が配置される。二ノ壇は幅12m×長さ30m余りで事実上の主郭。城内には磨崖仏が彫られた巨石が点在する。切岸や堀切の遺構が残り、眺望も良好。

#### アクセス

- **電車**: JR山陽本線鴨方駅より徒歩40分
- **車**: 山陽自動車道鴨方ICから6分
- **駐車場**: 登城口前に無料（約5台）
- **登城路**: 麓の長川寺から徒歩。北側尾根沿いの道路で車進入も可能

#### 野州家の特徴

- **領国**: 備中浅口郡・小田郡、伊予宇摩郡・温泉郡
- **性格**: 管領細川京兆家の分家で、本家を補佐する立場
- **家名の由来**: 2代持春と3代教春が連続して「下野守」（しもつけのかみ＝野州）を称したことから「野州家」と呼ばれる
- **京兆家との関係**: 4代政春の子・細川高国が京兆家当主となり管領に就任。野州家は幕府権力の中枢にも関与した

### 2. 赤澤修理亮の正体

#### 「修理亮」の官位の意味

「修理亮」（しゅりのすけ）は律令制における修理職の次官で、従五位下に相当する。修理職は宮中の建物の修理・造営を司る役所だが、室町時代には実際の職務とは無関係に武家が名乗る「官途名」として使用された。従五位下以上は「通貴」（貴族の位階）とされ、修理亮は地方武将としてはかなりの格式を示す。

#### 赤澤修理亮に関する史料

**『備中集成誌』の記載**：
> 「建武年中藤田小次郎住して、薬師寺公義に与力する。小次郎は新田義貞十六騎発、藤田三郎左衛門が弟なりという。応仁年中鴨方細川氏の重臣、赤澤修理亮住す。その本国は信州なり。」

**『倉敷市史 第2巻 古代・中世』の記載**：
- 細川氏方の赤澤修理亮の名が記されている

#### 赤澤修理亮の考察

| 項目 | 分析 |
|------|------|
| **活動時期** | 応仁年間（1467〜1469年） |
| **居城** | 森本松山城（鴨山城の支城） |
| **身分** | 鴨山城主細川氏の重臣 |
| **出自** | 「本国は信州なり」＝信濃出身の赤沢氏 |
| **系統** | 赤沢本流（小笠原長清→清経の系統）の可能性が高い |

**本流のどの系統に属するか（推定）**：

応仁年間（1467〜69年）に信州から備中に移動した赤沢氏の系統を考えると、以下の可能性がある。

1. **赤沢満経の系統**: 信濃守護代を務めた赤沢満経（応永7年/1400年の大塔合戦に登場）の子孫。応永19年（1412年）に細川氏の使者を務めた「赤沢三郎」は、この時期の満経一門の人物と考えられる。

2. **赤沢朝経以前の細川被官**: 赤沢朝経（永正4年/1507年没）が有名だが、朝経以前から赤沢氏と細川氏の関係は確認されており、応永19年（1412年）の赤沢三郎が最古の記録。この赤沢三郎の子孫が修理亮である可能性がある。

3. **赤沢氏の「修理」系統**: 備中赤沢氏では「修理」の官職名が代々使われた形跡がある（修理亮→後年の宗連が「修理進」を称す）。「修理」は赤沢氏の一系統を示す可能性がある。

**重要な推定**: 赤澤修理亮は、応仁の乱（1467年〜）の混乱期に、3代目鴨山城主・細川教春（西軍に属した）の重臣として信濃から従って備中に入国した人物と考えられる。教春は応仁の乱で西軍に属しており、この時期に家臣団の再編・増強があったはずで、信濃の赤沢氏がこの機に細川野州家に仕えた可能性は十分にある。

### 3. 赤澤三郎四郎朝宗の考察

#### 基本情報

| 項目 | 内容 |
|------|------|
| **名前** | 赤澤三郎四郎朝宗（あかざわ さぶろうしろう ともむね） |
| **仕えた主君** | 細川石高馬賢（持賢）（1441年生・満元の子の嫡の嫡） |
| **時代** | 15世紀後半（応仁期の前後） |

#### 「赤澤三郎四郎」の通称の意味

- **「三郎四郎」**は複合通称で、「三郎」の家の「四郎」（四男または四番目の系統）を意味する。つまり赤沢三郎家の四代目、あるいは三郎の四男ということを示す可能性が高い。
- 応永19年（1412年）に細川氏の使者を務めた**「赤沢三郎」**は、朝宗の先代にあたる人物と推定される。赤沢三郎が細川被官として活動し、その子（四男？）が「三郎四郎」と称して朝宗を名乗ったという解釈が可能。
- 時代差は約30〜40年で、親子または祖父・孫の関係として整合する。

#### 「朝宗」の名前 — 赤沢朝経との「朝」の字の共通性

**「朝」の字の分析**：

赤沢氏の系譜において「朝」の字を持つ人物：
- **赤沢朝経**（?〜1507年）：細川政元の重臣。赤沢氏の中で最も著名な人物
- **赤澤三郎四郎朝宗**：細川持賢に仕えた人物
- **赤沢朝治**：政経の子か（冊子記載）

赤沢氏の主な通字は「経」（清経→安経→経顕→氏経→経光→武経→満経→経隆→**朝**経→政経→**経**智→貞**経**）だが、「朝」の字は朝経・朝宗・朝治の3人に確認される。

**考察**: 「朝」の字は、赤沢氏の中でも特に細川氏との関係が深い系統が使用した通字である可能性がある。朝宗が細川持賢に仕えていたこと、朝経が細川政元に仕えていたことは、「朝」を通字とする一族が細川被官の家系であったことを示唆する。

**朝宗と朝経の関係（推定）**：
- 朝宗は朝経と同時代かやや先行する人物。朝宗が仕えた細川持賢（1441年生）は野州家の分流ではなく典厩家・京兆家系統の人物。
- 朝経の養子となった赤沢長経の例から、赤沢氏の嫡流と庶流が異なる細川氏の分家に仕えていた構造がうかがえる。
- **朝宗は朝経の同族（おそらく兄弟か従兄弟）**で、朝経が京兆家の直臣として中央で活躍したのに対し、朝宗は典厩家/野州家系統に仕えて備中方面で活動した可能性がある。

#### 細川持賢（石高馬賢）との関係

- 「細川石高馬賢」は、備中赤澤氏サイトの表記で、正しくは**細川持賢**（もちかた）の可能性が高い（原文の読み取りに難がある部分）。
- 細川持賢は細川満元の孫で、典厩家の当主。京兆家を補佐する立場にあった。
- 赤沢氏の被官としての活動は、京兆家（本家）だけでなく、典厩家・野州家など細川氏の各分家にも及んでいた。

### 4. 大坂の陣と備中赤澤氏

#### 伝承の内容

備中赤澤氏サイトの管理人zawa氏による記述：

> 「完全余談だが、赤澤一族は関ヶ原で改易となり、それ以前の豊臣家の恩顧に報いるために備中より大坂に入城し、真田隊に一族上げて所属し多くが討ち死にした、と伝えられている。」

#### 関ヶ原で改易された経緯

1. **慶長5年（1600年）**: 関ヶ原の戦いで、細川元通（鴨山城主）は毛利氏に従い西軍に属した
2. **西軍の敗北**: 毛利氏は周防・長門の2ヶ国に大減封
3. **鴨山城の喪失**: 細川元通は鴨山城を失い、毛利秀元のもとへ身を寄せた
4. **赤沢氏の分散**:
   - 宗連（修理進）は細川元通に従い長州に移住
   - 宗台（久助宗白？）は関ヶ原で戦死（近江国膳所の赤沢谷で討死）
   - **残りの一族は備中に残留し、領地を失って「改易」状態になった**

#### 関ヶ原から大坂の陣までの15年間（推定）

関ヶ原（1600年）から大坂冬の陣（1614年）まで15年間、備中に残った赤澤一族がどこにいたかについて：

- 備中浅口郡の柏島・玉島地区に在地の庄屋・郷士として残留した可能性が最も高い
- 森本松山城の城下である柏島は赤澤氏の本拠地であり、城を失っても土地に残ることは自然
- 「関ヶ原で改易」とは、細川元通が鴨山城を失ったことに伴い、赤澤氏も武士としての身分・知行を失ったことを意味する
- 14〜15年の浪人生活の間に、かつての主家の恩顧を返すべく大坂方への参陣を決断したと推定される

#### なぜ真田隊に属したのか（推定）

真田信繁（幸村）と備中赤澤氏の接点について考えられるルート：

1. **信濃つながり**: 赤澤氏の「本国は信州」であり、真田氏も信濃国の武将。信濃の同郷意識から真田隊に合流した可能性がある。特に赤澤氏は小笠原氏の庶流であり、信濃の武家ネットワークに属していた。

2. **高野山つながり**: 真田昌幸・信繁父子は関ヶ原後に高野山に蟄居した。赤沢宗伝（阿波）も上桜城の戦い後に高野山に3年間籠った例があり、赤沢一族と高野山の縁は深い。備中赤澤氏が高野山を通じて真田氏と接触した可能性もある。

3. **浪人ネットワーク**: 大坂の陣では全国から約10万の浪人が集結した。関ヶ原で改易された武士たちの情報網は全国に張り巡らされており、備中赤澤氏もこのネットワークを通じて大坂入城の情報を得たと考えられる。

4. **豊臣家への恩顧**: 細川元通は毛利氏を通じて豊臣政権に従っていた。赤澤氏も豊臣家への恩義を感じていた可能性がある。

#### 大坂の陣での具体的な行動（推定）

- **大坂冬の陣（慶長19年/1614年11〜12月）**: 真田信繁は大坂城の南側に「真田丸」を築き、ここで徳川軍に大損害を与えた。赤澤一族が真田隊に属していたなら、真田丸の防衛戦に参加した可能性がある。

- **大坂夏の陣（慶長20年/1615年5月）**: 真田信繁は天王寺・岡山の戦いで家康本陣に突撃し、戦死した。赤澤一族も最終決戦で多くが討死したと考えられる。真田隊は最も激しい戦闘を行った部隊であり、「多くが討死」という伝承と整合する。

#### 「多くが討死」の規模（推定）

- 森本松山城主の赤澤氏は、3代にわたり城主を務めた有力家臣で、一定規模の一族郎党を擁していたはず
- 関ヶ原時点で戦闘員10〜30人程度の一族であった可能性がある（城主級の武家としては妥当な規模）
- 「多くが討死」とは、この一族の過半数が大坂で命を落としたという意味と推定される
- 真田隊全体の戦死率から推定すると、一族の7〜8割が討死してもおかしくない

#### 伝承の出典

この伝承はzawa家の**家伝**（口伝または家系図の添え書き）に基づくものと推定される。zawa家の家系図には「宇多天皇第九皇子教實王」からの系譜が記されており、家伝が代々受け継がれてきたことがうかがえる。学術論文や公的史料での裏付けは確認されていないが、家伝として代々語り継がれてきた伝承は、一定の歴史的核を持つ可能性が高い。

### 5. 玉島合戦（元亀2年/1571年）の詳細

#### 基本情報

| 項目 | 内容 |
|------|------|
| **時期** | 元亀2年（1571年）2月 |
| **場所** | 備中国浅口郡玉島および片島付近 |
| **背景** | 尼子家再興のため尼子勝久・山中鹿之助幸盛らが蜂起。打倒毛利の一環として備中に侵攻 |

#### 両軍の兵力

| 陣営 | 兵力 | 本陣 |
|------|------|------|
| **尼子勢** | 7,000余 | 倉敷市玉島付近 |
| **細川勢** | 2,000余（侍150騎、雑兵2,000余） | 浅口市鴨方町杉山城 |

#### 細川勢の主な武将

| 武将名 | 結果 |
|--------|------|
| 細川下野守通薫（通董） | 生還 |
| **赤澤五郎四郎宗照** | **戦死？** |
| 中塚二郎兵衛季龍 | 戦死 |
| 中塚治左衛門季信 | 戦死 |

#### 戦いの経過

1. 尼子勝久・山中鹿之助幸盛らが出雲で蜂起し、打倒毛利を掲げる
2. 九州討伐のため手薄になった毛利方の備中に、尼子勢が数千の兵を率いて侵攻
3. 毛利方の庄伊豆守高貫らも抵抗したが尼子方に降伏し、先手となって細川領に押し寄せた（総勢7,000）
4. 対する細川勢はわずか2,000
5. 尼子勢は細川勢の城を2〜3攻略し、細川通董のいる鴨方杉山城に迫った
6. 細川勢は杉山城から倉敷の片島まで撃って出て、要害を背に戦うも敗北
7. 敗残兵は総社幸山城に逃げ込み、そこで抵抗を継続
8. 毛利元就の4男・毛利元清が8,000の兵を連れて反撃し、尼子勢は出雲まで退却

#### 赤澤五郎四郎宗照の戦死について

備中赤澤氏サイトでは、宗照の戦死に「？」がつけられており、確定的ではない。時系列的な問題もある：

- 宗照は細川通政に仕えたとされるが、通政は天文5年（1536年）に尼子に圧迫されて伊予に退いている
- 永禄2年（1559年）に通董が帰還した際に宗照がこれを迎えたという記述もある
- 元亀2年（1571年）の玉島合戦で戦死したとすると、1559年から12年間は通董に仕えていたことになる
- 宗照の没年を1570年頃とする説もあり、玉島合戦（1571年）の直前に別の戦いで討死した可能性もある

#### 信濃出身の武将の参戦

この戦いに信濃出身の武将が参戦した記録は確認できていないが、赤澤五郎四郎宗照自身が「本国は信州」の赤澤氏であり、まさに信濃出身の武将が細川勢として参戦していたことになる。

### 6. 森本松山城の詳細

#### 基本情報

| 項目 | 内容 |
|------|------|
| **所在地** | 備中国浅口郡柏島（現・岡山県倉敷市玉島柏島字溝所） |
| **城の性格** | 鴨山城の支城・出城。海上交通の監視拠点 |

| **城主** | 赤澤宗春→赤澤宗照→赤澤宗栄（3代） |

#### 城の構造

| 要素 | 詳細 |
|------|------|
| **立地** | 玉島柏島の海に面した丘陵上 |
| **小字名** | 溝所（みぞしょ）※「政所」（まんどころ）と同音の可能性 |

- この二度にわたる養子相続により、板西城の赤沢氏は血統的には必ずしも一直線ではないが、「赤沢」の家名と所領は連続して維持された

### 3. 3,000貫の意味

#### 貫高制における3,000貫

| 換算基準 | 3,000貫の石高換算 |
|----------|-------------------|
| 1貫＝2石（一般的換算） | **6,000石** |
| 1貫＝5石（別の換算） | **15,000石** |
| 1貫＝10石（最大換算） | **30,000石** |

学術的には、戦国期の阿波国では1貫＝2〜5石程度と推定されるため、**3,000貫は6,000〜15,000石**に相当する。

#### 兵力動員力

天文年間（1532〜1555年）の換算では、7貫文につき1人の兵力動員が目安とされる。
- 3,000貫 ÷ 7 ≒ **約430人**の常備兵力
- 中富川の戦いで宗伝が率いた2,000の兵は、一族郎党・農兵・傭兵を含めた総動員数

#### 阿波国内での位置づけ

| 勢力 | 知行 | 備考 |
|------|------|------|
| 三好氏（全体） | 阿波一国（約18万石） | 守護代→事実上の守護 |
| 赤沢宗伝 | 3,000貫（6,000〜15,000石相当） | 阿波国内の主要家臣 |
| 十二人衆各城主 | 50〜100貫（100〜500石相当） | 宗伝の配下 |

3,000貫は阿波国の約3〜8%に相当し、三好家中でも有数の大勢力であった。板野郡の広い領域を支配し、十二の支城を配下に置くだけの規模を持っていた。

参考として、毛利元就が国人領主時代に持っていた領地が3,000貫であり、これは一国の守護にはなれないが、地方の有力国人としてはかなりの規模であった。

**安芸飛騨守**：
- 赤沢宗伝の**娘婿**（宗伝の娘と結婚）
- 十二人衆の中では河内城・安芸宗長（100貫）と同一人物の可能性がある
- 中富川の戦いで生き延びた
- **戦後、板野町犬伏にて帰農**。武士を捨てて農民として暮らした
- 板西城三人衆の中で唯一生存が確認される人物

#### 三人衆の役割

三人衆は十二人衆とは別格の上位重臣で、宗伝を補佐して3,000貫の勢力全体の運営にあたった。軍事面では各方面軍の指揮を分担し、行政面では領内の徴税・裁判などを担ったと推定される。安芸飛騨守が宗伝の娘婿であったことは、三人衆が単なる家臣ではなく、一族として宗伝と結びついていたことを示す。

## 第三部：備中赤澤氏と阿波赤沢氏の比較・関連性

---

### 両氏の比較

| 比較項目 | 備中赤澤氏 | 阿波赤沢氏 |
|----------|------------|------------|
| **出自** | 清和源氏小笠原氏庶流 | 清和源氏小笠原氏庶流 |
| **備中/阿波入国時期** | 応永14年（1407年）頃 | 鎌倉時代（13世紀） |
| **入国の経緯** | 細川氏家臣として | 近藤六親家の養子として |
| **仕えた主家** | 細川野州家 | 三好氏（→十河存保） |
| **居城** | 森本松山城（鴨山城の支城） | 板西城 |
| **家紋** | 丸に三階菱 | 松皮菱の内に十文字（故城記記載） |
| **最盛期の規模** | 支城1城の城主 | 3,000貫、12支城を統括 |
| **滅亡の契機** | 関ヶ原の戦い（1600年） | 中富川の戦い（1582年） |
| **滅亡後** | 大坂の陣で真田隊に属し討死（家伝） | 正本法師が勝覚寺を開基 |

### 両氏の関連性

備中赤澤氏サイトでは、阿波の赤沢氏が長宗我部氏の四国統一戦に敗れ、備中に逃亡して土着したという**仮説③**（阿波守護小笠原氏一族としての流入説）が紹介されている。しかしサイト管理人自身がこの説の根拠は薄いとし、**仮説①の細川氏家臣説が最も可能性が高い**としている。

とはいえ、中富川の戦い（1582年）で板西城が落城した後、讃岐に逃れた一族がさらに備中に渡った可能性は完全には否定できない。備中赤澤氏と阿波赤沢氏が直接的な親子関係にある可能性は低いが、同じ小笠原氏庶流として遠い同族であることは確実である。
