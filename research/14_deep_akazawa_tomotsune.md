## 1. 出自と若年期

### 赤沢氏の系譜

赤沢氏は清和源氏の流れを汲む名門で、以下の系譜を持つ。

```
清和天皇 → 源経基 → 源満仲 → 源頼信 → 源義光（新羅三郎）
→ 武田氏 → 加賀美遠光 → 小笠原長清 → 長経
→ 清経（伊豆国田方郡赤沢郷に住し、赤沢氏を称す）
→ 安経 → 経顕 → 氏経 → 常興 → 経光 → 武経
→ 満経 → 経隆 → 【朝経】
```

### 父・赤沢経隆との関係

- 父・経隆は赤沢氏本流の当主で、通称は天真郎、官位は伊豆守。
- 「小笠原」姓を併称しており、本家小笠原氏との密接な関係を示す。
- 朝経が家督を嫡子・政経に譲った後、経隆は朝経・弟の幸純とともに上洛している。
- つまり、朝経一族は父子そろって信濃を離れ、畿内での活動に転じた。

### 信濃から上洛した経緯

- 朝経の上洛の正確な時期は不明だが、延徳3年（1491年）の六角高頼征伐への従軍が、史料上で確認できる最初の畿内での活動である。
- 上洛の背景には、小笠原氏内部の争い（小笠原宗康と小笠原光康の間の信濃守護職争い）で、赤沢氏が信濃での立場を失った可能性がある。
- 教経（満経の系統）が鎌倉公方に仕えて小笠原家の内紛で戦死しており、赤沢氏一族は15世紀半ばに大きな転機を迎えていた。

### 小笠原流弓馬術の習得

- 赤沢氏は初代・清経以来、小笠原氏庶流として弓馬術の伝統を受け継いでいた。
- 朝経は「糾法（弓法）的伝」（小笠原流弓馬術の核心的な伝承）を本家小笠原氏に倣って修得。
- この弓馬術の技能が、後に細川政元への仕官、さらに将軍の弓道師範という地位を得る決定的な契機となった。

## 2. 細川政元との関係

### 仕官の経緯

朝経が細川政元に仕えた経緯は、二つの契機が史料に記されている。

**第一の契機：弓馬術の伝授**
- 朝経は小笠原流の「糾法（弓法）的伝」を管領・細川政元に伝授した。
- これが縁となり、政元に重用されるようになった。

**第二の契機：六角高頼征伐と鷹狩**
- 延徳3年（1491年）、10代将軍・足利義稙（当時は義材）の六角高頼征伐に従軍。
- この際、朝経の鷹狩の技能が政元に認められ、正式にその被官（家臣）となった。
- 朝経は「外様の内衆」として細川京兆家に組み込まれた。

### 内衆としての地位と役割

「内衆」とは、細川京兆家の家中における中核的家臣団のことで、外様の内衆は本来の京兆家譜代ではないが、実力によって中枢に取り立てられた者を指す。朝経が就いた要職は以下の通り。

| 役職 | 内容 |
|------|------|
| 山城上三郡守護代 | 山城国の上三郡（葛野郡・愛宕郡・紀伊郡）の実質的な統治者 |
| 幕府料所・河内十七ヵ所給人 | 幕府直轄領である河内国17ヵ所の管理・徴税責任者 |
| 妙法院領河内八ヵ所代官 | 寺社領の代官として実質的な支配を行う |
| 武者所 | 将軍直属の武官職 |
| 山城国槙島の統治者 | 山城国槙島（現・京都府宇治市槙島町）を本拠地として統治 |

朝経は政元政権の「軍事的柱石」と評され、政元の「手足」として畿内各地での軍事作戦を主導した。

学習院大学の家永遵嗣教授は、朝経を「宗瑞（北条早雲＝伊勢宗瑞）を圧迫した立役者」と評しており、朝経の軍事力が当時の関東情勢にまで影響を及ぼしていた可能性を指摘している。

### 政元の修験道信仰との関係

- 細川政元は修験道に深く傾倒し、「出家のごとし、山伏のごとし」と称された異色の管領であった。
- 女人禁制（不犯）の戒めを厳守し、生涯独身を通したため、実子がおらず3人の養子（細川澄之・細川澄元・細川高国）を迎えた。
- 朝経と政元の修験道信仰との直接的な関係を示す史料は確認できないが、朝経が比叡山や大和の寺社を躊躇なく焼き討ちできた背景には、政元の既存宗教権威に対する反発が影響していた可能性がある。

### 学術論文

- 「細川政元政権と内衆赤沢朝経」（CiNii Research所収、CRID: 1520853832269358720）
  - 朝経の内衆としての地位と役割を詳細に論じた専門論文。

## 3. 比叡山焼き討ち（1499年・明応8年7月11日）

### 背景

- 明応の政変（1493年）で将軍職を奪われた足利義稙（義尹）は、京を脱出して越中に入り、越中守護代・神保長誠に迎えられた。
- 義稙は加賀守護・富樫泰高、越前の朝倉貞景の支持を得て上洛の姿勢を示した。
- 比叡山延暦寺は義稙を支持し、反政元派に回った。
- さらに、反政元派の幕臣・桃井政信と公家の西坊城顕長、延暦寺の僧・東蔵坊が挙兵した。

### 戦闘の経過

- 明応8年（1499年）7月11日、細川政元は内衆の赤沢朝経と波々伯部宗量に延暦寺焼き討ちを命令。
- 朝経と宗量の率いる軍勢は、早朝から一斉に比叡山に攻め上った。
- まず桃井政信、西坊城顕長、東蔵坊を討ち取った。
- 続いて山上の主要伽藍を悉く焼き払った。

### 焼失した建造物

| 建造物 | 備考 |
|--------|------|
| 根本中堂 | 延暦寺の中心堂宇、天台宗の総本堂 |
| 大講堂 | 学問・論議の中心施設 |
| 常行堂 | 常行三昧の修行道場 |
| 法華堂 | 法華経を読誦する道場 |
| 延命院 | 僧坊 |
| 四王院 | 四天王を祀る堂宇 |
| 経蔵 | 経典を収蔵する施設 |
| 鐘楼 | 梵鐘を吊るす建物 |

### 規模と被害

- 山上の主要伽藍がことごとく焼失するという壊滅的被害。
- これは織田信長による元亀2年（1571年）の比叡山焼き討ちに先立つこと**72年前**の出来事。
- 信長以前に延暦寺を徹底的に破壊した武将として、赤沢朝経は特筆される。

### 朝廷・寺社の反応

- 比叡山は「鎮護国家」の象徴であり、その焼失は朝廷・寺社社会に衝撃を与えた。
- しかし、政元の権力の前に、朝廷も有効な対抗策を講じることはできなかった。
- 大乗院門跡の尋尊は『大乗院寺社雑事記』に「先代未聞沙汰也」（前代未聞の事態である）と記している。

## 4. 第一次大和侵攻（1499年・明応8年9月〜12月）

### 山城南部の制圧（9月）

- 比叡山焼き討ちの後、明応8年9月、河内で義稙派として挙兵した畠山尚順に呼応した筒井順賢・十市遠治ら大和国人衆を牽制するため、朝経は山城南部へ急行。
- 『後法興院政家記』明応8年9月26日条に「早朝より宗益（朝経）、御牧城を攻め落とし、放火す。首9つ討ち取ると云々」と記される。
- **御牧城**・**水主城**・**槙島城**を次々に陥落させた。

### 大和侵入（12月）

- 明応8年12月18日、朝経は歌姫街道（現・奈良市北部）を経由して大和に侵入。
- 京都から大和への進軍ルートは、歌姫街道を通り、超昇寺城方面へ向かうものであった。

### 超昇寺城の戦い

- 超昇寺城で筒井党と合戦。
- 朝経が圧勝し、筒井党は多数の死傷者を出して敗走。
- 戦闘はおよそ6時間にわたり、午前10時頃から午後4時頃まで続いた。

### 焼き討ちされた寺社

| 寺社名 | 被害状況 |
|--------|----------|
| 法華寺 | 打ち壊し |
| 海龍王寺 | 打ち壊し |
| 西大寺 | 打ち壊し |
| 喜光寺 | 放火。鎌倉時代再興の本堂を始めとする堂塔伽藍がほぼ全焼 |
| 額安寺 | 焼き討ち |

### 奈良乱入

- 古市澄胤の手引きにより奈良に乱入。
- 興福寺の院下・僧房から春日大社まで、略奪と乱暴狼藉の限りを尽くした。
- 大和北部を占領し、尚順派の国人の所領を没収。

### 「大乗院寺社雑事記」での記述

- 門跡・尋尊（一条兼良の子、1430〜1508年）は、自らの本拠地である興福寺周辺の破壊に衝撃を受け、「先代未聞沙汰也」と記した。
- 尋尊の日記は約60年間（1450〜1508年）にわたる貴重な同時代史料で、朝経の大和侵攻を直接目撃した証言である。

## 5. 明応9年〜文亀年間の活動（1500年〜1503年）

### 大和での所領支配強化（1500年）

- 明応9年（1500年）、朝経は第一次侵攻で獲得した大和の所領支配をさらに強化。
- 尚順派の国人衆の残党を掃討。

### 畠山尚順撃破（1500年9月）

- 明応9年9月、河内で畠山尚順を撃破。
- 政元の畿内支配の拡大に貢献。

### 文亀2年の大和出兵（1502年）

- 文亀2年（1502年）、興福寺からの撤兵要求を受けたにもかかわらず、平然と大和に出兵。
- これは、朝経が大和を恒常的な支配領域として位置づけていたことを示す。

### 近江出兵（1503年3月）

- 文亀3年（1503年）3月、義稙派に接近した近江守護・六角高頼に追い込まれた義澄派の守護代・伊庭貞隆が挙兵。
- 朝経は近江に出兵し、伊庭貞隆と共に六角高頼が籠城した蒲生氏の音羽城を攻撃した。
- これは朝経の活動範囲が大和・河内だけでなく近江にまで及んでいたことを示す。

## 6. 薬師寺元一との共謀と赦免（1504年）

### 反乱の経緯

- 永正元年（1504年）3月、朝経は細川政元に対して反逆。
- 同年9月、摂津守護代・薬師寺元一と共謀し、政元の廃立を企てた。
- 元一は弟と共に淀城で挙兵。政元の養子・細川澄元を擁立して政権奪取を図った。
- しかし、元一の弟・薬師寺長忠らの攻撃を受けて敗北。元一は捕らえられ、京都に送られて翌日自刃（享年28歳）。

### 赦免

- 朝経も捕らえられたが、**その豪勇を政元から惜しまれて助命**された。
- 翌永正2年（1505年）、罪を許され、山城上三郡守護代に復帰。

### 歴史的意義

- この事件は、朝経と政元の関係が単純な主従関係ではなく、時に対立を含む複雑なものであったことを示す。
- 一方で、反乱を起こしても助命されたことは、政元にとって朝経の軍事力が不可欠であったことの証明でもある。

## 7. 河内侵攻（1506年・永正3年1月）

### 背景

- 畠山義英と畠山尚順は、1504年12月に和睦し、河内を分割統治していた。
  - 義英 → 誉田城（こんだじょう）
  - 尚順 → 高屋城（たかやじょう）

### 戦闘の経過

- 永正3年（1506年）正月、朝経は政元の命を受けて河内に侵攻。
- **畠山義英の居城・誉田城**を陥落させた。
- 続いて**畠山尚順の居城・高屋城**を陥落させた。

### 戦後の支配体制

- 両畠山氏を紀伊方面へ追放。
- 河内を細川政元の勢力圏に組み込んだ。
- 嫡子・政経が河内の防衛にあたった（小野上城・高屋城の守備）。
- 朝経自身は幕府料所・河内十七ヵ所の給人として河内の徴税・行政を担った。

## 8. 第二次大和侵攻（1506年・永正3年7月〜9月）

### 侵攻の背景

- 大和国人衆が畠山攻めへの協力を拒否したこと。
- 紀伊に逃れた両畠山氏を大和国人が匿ったことを罪に問う。
- 大和国人衆は1505年2月に春日社頭で起請文を捧げ、局外中立と他国衆排除を誓う「国人一揆」を結成していたが、朝経はこれを無視した。

### 大和国人一揆の形成

- 筒井氏、越智氏、十市氏ら、本来対立関係にあった国人たちが朝経の脅威を前に団結。
- 春日大社の社頭で起請文を交わし、一揆を結成。
- しかし、朝経の圧倒的軍事力の前に敗退を重ねた。

### 戦闘の詳細な経過

| 日付 | 出来事 |
|------|--------|
| 永正3年7月 | 古市澄胤の手引きで大和に侵入 |
| 8月1日 | 筒井順賢が朝経への助命嘆願を拒否し、国人一揆に参加 |
| 8月3日 | 西ノ京を陥落。郡山を陥落 |
| 8月4日 | 郡山に陣を進める。井戸城で筒井順賢は東山中に撤退 |
| 8月11日 | 戒重城を陥落 |
| 8月16日 | 筒井順賢が古市郷を急襲し、郡山城を一時奪回 |
| 8月24日 | **布留郷の決戦** — 筒井順賢と決戦。朝経大勝。筒井方は数十人の有力家臣を失い、吐山まで撤退 |
| 8月27日 | 耳成山に陣を構え、多武峰攻撃を開始 |
| 9月5日 | **多武峰（妙楽寺）を陥落** |

### 焼き討ちされた寺社（第二次侵攻）

| 寺社名 | 被害状況 |
|--------|----------|
| 法華寺 | 再度の焼き討ち |
| 菩提山正暦寺 | 焼き討ち |
| 多武峰妙楽寺 | 陥落・焼き討ち |
| 龍門寺 | 焼き討ち |

### 戦後処理

- 討ち取った国人衆の**首級131**を京都に送った。
- 闕所（没収）となった国人衆の所領を自らの支配下に収めた。
- **奈良盆地一帯を知行地化**し、大和を事実上の赤沢氏の支配領域とした。
- 三好之長（後の三好長慶の祖父にあたる）と協力して支配体制を構築。

## 9. 弓道師範としての活動

### 足利義政との関係

- 朝経は小笠原流弓馬術（糾法的伝）の正統な伝承者として、8代将軍・足利義政の**弓道師範**に就任。
  - ※一部の史料では「足利義尚」の弓道師範とする記述もあるが、義政説が通説。
- **武者所**（将軍の近衛武官）を兼任。
- 将軍の弓道師範という地位は、朝経に宮中・幕府での高い身分を与え、細川政元への仕官の後ろ盾ともなった。

### 小笠原流の伝承

- 赤沢氏は初代・清経以来、小笠原氏庶流として弓馬術を代々伝承していた。
- 朝経は政元に弓術を伝授したが、これは単なる技術指導ではなく、「糾法的伝」という小笠原流の核心的な秘伝を含むものであった。
- この伝承は、後に朝経の孫・経智を通じて近世の小笠原流（弓馬術礼法小笠原教場）へと受け継がれる。

### 鷹狩の技能

- 朝経は弓馬術だけでなく、鷹狩にも優れた技能を持っていた。
- 延徳3年（1491年）の六角高頼征伐の際、この鷹狩の技能が政元に認められたことが、被官化の直接の契機となった。
- 出家後に残した伝書『宗益相伝書』は鷹狩に関するものであり、弓馬術と鷹狩の両方に通じた文武両道の人物であったことがわかる。

### 弓道の技法について

- 小笠原流弓馬術は、歩射（徒歩での弓射）と騎射（馬上での弓射＝流鏑馬など）の両方を含む総合的な弓馬術。
- 「糾法的伝」とは、弓術の正しい射法・作法の伝授を意味し、小笠原流の根幹をなすもの。
- 具体的な技法としては、射法（射る際の作法・身体の使い方）、的伝（的を射る技術の伝授）、礼法（弓術に伴う礼儀作法）が含まれる。

## 10. 永正の錯乱と最期（1507年・永正4年）

### 背景：丹後出陣

- 永正4年（1507年）、朝経は細川政元の命を受け、丹後国の**一色義有（義季）**を攻撃するために出陣。
- 丹後は一色氏の領国であり、政元は丹後を直接的な勢力圏に組み込もうとしていた。

### 細川政元暗殺の詳細（6月23日）

**暗殺の背景**
- 政元には3人の養子がいた。
  - **細川澄之**（九条政基の子、関白・近衛尚通の猶子）
  - **細川澄元**（阿波細川氏出身）
  - **細川高国**（細川政春の子、野州家出身）
- 当初、澄之が後継者と目されていたが、後に澄元が台頭し、澄之は廃嫡の危機に立たされた。
- 澄之派の内衆・**香西元長**と**薬師寺長忠**が、政元排除を画策。

**暗殺の状況**
- 永正4年6月23日夜、政元は月待ちの行水中（邸内の湯残に入浴中）であった。
- 香西元長・薬師寺長忠に唆された**竹田孫七**らが、浴室に押し入り政元を殺害。
- 政元、享年42歳。

### 暗殺当日の朝経の行動

- 朝経は暗殺当時、丹後国で一色義有と対陣中であった。
- 6月26日（一説には6月27日）、主君・政元暗殺の報が朝経のもとに届いた。
- 朝経は直ちに軍を京都に撤退させようと決断。

### 丹後九世戸への逃亡と最期

**反撃の開始**
- 政元の死を知った丹後の国人衆が一斉に蜂起。
- **一色義有**（朝経が攻撃していた相手）が反撃を開始。
- **石川直経**ら丹後の国人衆も朝経の軍に攻撃を加えた。
- 朝経の軍は包囲される形となった。

**文殊堂での自刃**
- 朝経は宮津付近で国人一揆に包囲された。
- **永正4年6月26日**、**丹後九世戸（くせど）の文殊堂**（現・智恩寺、天橋立の南端）にて**自刃**。
- 享年57歳。
- 智恩寺は「切戸の文殊」「九世戸の文珠」とも呼ばれ、日本三文殊の一つ。天橋立の南端に位置する名刹。

**同行者の運命**
- 若狭国の**粟屋親栄**も討ち取られた。
- **古市胤盛**（古市澄胤の子、かつて朝経の大和侵攻に協力した古市澄胤の息子）も討死。
- 弟で養子の**赤沢長経**は奇跡的に生還し、その後細川澄元の部将として仕えた。

### 『寛政重修諸家譜』の異説

- 『寛政重修諸家譜』では、朝経の自刃の日付を**7月4日**としており、Wikipediaなどで通説とされる6月26日と異なる。
- この差異は、情報伝達の時間差や史料の記載方法の違いによるものと考えられる。

## 11. 「澤蔵軒宗益」の号について

### 入道の経緯

- 朝経は家督を嫡子・政経に譲った後、入道（出家）して「**澤蔵軒宗益**（たくぞうけん そうえき）」と号した。
- 入道の正確な時期は不明だが、延徳3年（1491年）の政元被官化の前後と推定される。

### 「澤蔵軒」の意味

- 「軒号」は禅宗（臨済宗・曹洞宗）において、大寺院の住職が隠居した際に住む寮舎（軒）の名に由来する。
- もとは大寺院の敷地内に建物を一棟寄進した信者に贈られるものであった。
- 「澤蔵」の「澤」は、赤**沢**氏の「沢」の字を含んでおり、一族の名を冠した号である可能性がある（※推定）。
- 「蔵」は仏教用語で「法蔵（仏法の蔵）」を意味する可能性もある。
- 現存する史料からは「澤蔵軒」の具体的な由来は明確に確認できない。

### 「宗益」の意味

- 「宗」は禅宗における法名（道号）に多く用いられる文字。仏法の「宗旨」を意味する。
- 「益」は「利益」「増益」を意味し、仏法の修行による功徳を表す。
- 全体として「仏法の宗旨を益する者」という意味を持つと推定される。
- 政元が修験道に傾倒していたことを考慮すると、朝経の出家も政元の宗教的環境の影響を受けた可能性がある。

### 使用時期

- 入道後は一貫して「澤蔵軒宗益」の号を使用。
- 同時代史料『後法興院政家記』でも「宗益」の名で記されている（例：明応8年9月26日条「宗益、御牧城を攻め落とし…」）。

## 13. 後世の評価

### 畿内の猛将としての評価

- 赤沢朝経は、細川政元政権の軍事的柱石として、畿内における圧倒的な戦績を残した。
- 比叡山焼き討ち（1499年）、二度にわたる大和侵攻（1499年、1506年）、河内侵攻（1506年）など、畿内全域を席巻する軍事行動は、戦国初期の武将としては類を見ない規模。
- 「悪太郎」の異名は、神仏を恐れぬ苛烈な戦い方を象徴している。

### 大和の人々からの恐怖

- 大和国人衆は朝経の脅威の前に、本来対立していた筒井氏・越智氏・十市氏が団結して「国人一揆」を結成するほどであった。
- にもかかわらず、朝経の軍事力の前に敗退を重ね、奈良盆地一帯が朝経の知行地と化した。
- 大和における朝経の支配は、京衆による外部からの支配であり、在地の国人にとっては屈辱的なものであった。

### 北条早雲（伊勢宗瑞）との関係

- 学習院大学の家永遵嗣教授は、朝経を「**宗瑞（北条早雲＝伊勢宗瑞）を圧迫した立役者**」と評している。
- 細川政元政権の軍事力が、朝経を通じて関東情勢にまで影響を及ぼしていた可能性が指摘されている。
- 朝経の死（1507年）と政元の暗殺は、結果的に伊勢宗瑞の関東での勢力拡大を容易にした側面がある。

### 現代の歴史研究での位置づけ

- 「細川政元政権と内衆赤沢朝経」（CiNii Research所収）をはじめ、近年の研究で朝経の歴史的重要性が再評価されている。
- 戦国期細川権力の研究（吉川弘文館刊）でも取り上げられている。
- 従来は「細川政元の家臣」として付随的に扱われることが多かったが、近年は朝経自身の主体的な活動と、畿内の戦国化に果たした役割が注目されている。
- 信長に先立つ比叡山焼き討ちの実行者として、日本史の転換点における重要人物として位置づけられつつある。

- **赤沢幸純**（弟）: 上洛時に同行。福王寺・大和守。

### 朝経の家臣団・協力者

| 人物 | 関係 |
|------|------|
| 波々伯部宗量 | 細川政元の内衆。比叡山焼き討ちで朝経と共に軍を率いた |

### 朝経の家臣団の特徴

- 朝経の軍事力の中核は、信濃から上洛した赤沢氏一族と、政元から配属された内衆で構成されていたと推定される。
- 大和侵攻では、現地の親政元派国人（古市澄胤など）を案内人・協力者として活用。
- 京都から遠征する「京衆」としての性格を持ち、在地勢力とは本質的に異なる外部勢力であった。

## 16. 関連する場所（ゆかりの地）

### 塩崎城（長野県長野市篠ノ井塩崎）

- 赤沢氏の本拠地。信濃国埴科郡に位置。
- 父・常興が中先代の乱（1335年）後に四宮荘の管理と築城を担って以来、赤沢氏の拠点。
- 朝経はここで生まれ育ち、後に上洛。
- 現在は城跡として遺構が残る。

### 槙島（京都府宇治市槙島町）

- 朝経が山城国での本拠地とした場所。
- 山城上三郡守護代として、この地を拠点に畿内での軍事・行政活動を展開。

### 比叡山延暦寺（滋賀県大津市）

- 明応8年（1499年）7月11日、朝経が焼き討ちした場所。
- 現在の延暦寺の建造物は、この焼き討ち後に再建されたもの。

### 大和国（奈良県）

- 朝経が二度にわたり侵攻した地域。
- 主要な戦場：超昇寺城、郡山、戒重城、布留郷、多武峰
- 焼き討ちされた寺社：法華寺、海龍王寺、西大寺、喜光寺、額安寺、正暦寺、多武峰妙楽寺、龍門寺

### 河内国（大阪府東部）

- 朝経が畠山氏を追放した地域。
- 主要な城：高屋城（大阪府羽曳野市）、誉田城、小野上城

### 智恩寺・文殊堂（京都府宮津市天橋立）

- **朝経終焉の地**。
- 永正4年（1507年）6月26日、ここで自刃。
- 「九世戸の文殊堂」として知られ、日本三文殊の一つ。
- 天橋立の南端に位置する。
- 現在も観光名所として多くの参拝者が訪れるが、朝経の自刃に関する案内板等の有無は未確認。

### 墓所

- 朝経の墓所の正確な場所は、現存する史料からは特定できていない。
- 丹後で自刃したため、丹後の地に埋葬された可能性があるが、詳細は不明。
- 信濃の塩崎城近辺に赤沢氏の菩提寺・墓地がある可能性もある。
