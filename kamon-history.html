家紋の変遷
七つの紋が語る、一族 1,100年の分岐と継承
松皮菱に十文字
赤沢氏の主紋 — 信濃から阿波、備中へと受け継がれた本流の紋章

家紋は、単なるシンボルではない。一族がどこで分かれ、何を受け継ぎ、どう生き延びたかを語る「血脈の地図」だ。

赤沢氏には、少なくとも7つの家紋が確認されている。主紋である「松皮菱に十文字」から、地域や時代によって枝分かれした紋の数々。それぞれの紋には、一族の選択と運命が刻まれている。

ORIGIN

系譜の起点 — 三階菱から松皮菱へ

すべての始まりは、小笠原氏の三階菱にある。清和源氏の流れを汲む小笠原氏が本紋として用いた三階菱。赤沢氏はその小笠原氏から分かれた一族であり、家紋もまた三階菱をベースに独自の変形を遂げた。

三階菱（小笠原氏）
清和源氏義光流の本紋。甲斐源氏「菱」の系譜
松皮菱（赤沢氏の基本形）
松の樹皮を菱形に図案化。三階菱の変形
松皮菱に十文字（赤沢氏の主紋）
松皮菱の中央に十文字を加えた、赤沢氏独自の紋

「菱」は甲斐源氏のシンボル。武田氏の「割り菱」「四つ割り菱」と、小笠原氏の「三階菱」、赤沢氏の「松皮菱」は同じルーツを持つ。つまり家紋だけを見ても、赤沢氏が甲斐源氏の血を引く一族であることが読み取れる。

TRANSITION

七つの家紋 — 分岐と変遷の物語

一族が各地に散り、時代が移るにつれて、家紋もまた変遷した。ここでは7つの家紋を、本流から分岐した順に辿る。

松皮菱の内に十文字

本流 / 信濃 → 阿波 → 備中

赤沢氏の主紋。小笠原氏の三階菱から派生し、松皮菱の中央に十文字を配した独自の紋。信濃から阿波、備中へと移った本流が使用した。

紋の意味: 松の樹皮を菱形に図案化。「菱」は甲斐源氏のシンボルであり、十文字は赤沢氏の独自性を示す。

五七桐

副紋 / 将軍家からの賜紋

赤沢氏の副紋として使用。足利将軍から賜った可能性が高い。本流の松皮菱と併用され、正式な場では使い分けられた。

紋の意味: 天皇から足利将軍へ、さらに臣下へと下賜される「賜紋」の典型。権威と忠誠の象徴。

鷹の羽

讃岐系 / 与田川口 → 丹生・三本松・白鳥

讃岐（香川県）に逃れた赤沢氏の系統が使用。本流の松皮菱から完全に離れた独自の紋であり、讃岐での新たな出発を象徴する。

紋の意味: 矢羽根に使う鷹の羽を図案化。武勇の象徴として武家に広く好まれた紋。

立沢瀉（たちおもだか）

讃岐系 / 鷹の羽と併用

讃岐に逃れた赤沢氏が、鷹の羽と並んで使用した紋。水辺の植物オモダカを図案化したもの。

紋の意味: オモダカは「面高」とも書き、「勝ち草」として武家に好まれた。葉の形が矢じりに似ることから武運の象徴。

三階菱

復姓系 / 赤沢 → 小笠原に復した貞経の家

赤沢氏から小笠原氏に復姓した貞経の系統が使用。つまり「赤沢」の名を捨て、本家の紋に戻ったことを意味する。

紋の意味: 小笠原氏の本紋。三重の菱形は格式の高さを示す。復姓とともに家紋も本家に回帰した。

三つ紅葉

勝覚寺（讃岐）/ 鷹司家からの下賜

讃岐の勝覚寺が、公家の鷹司家から下賜された紋。赤沢氏の菩提寺としての格式を示す。

紋の意味: 紅葉は秋の風情を象徴し、平安貴族にも好まれた雅な紋。鷹司家からの下賜は寺院の格式の高さを物語る。

菱紋

筑摩郡稲倉 / 武田氏に降伏した系統

信濃国筑摩郡稲倉の赤沢氏が使用。武田氏に降伏した系統であり、家紋もより素朴な単純菱に変わっている。

紋の意味: 菱形は甲斐源氏の最も基本的な紋。武田氏への降伏後、本家の複雑な紋からシンプルな菱に回帰した可能性がある。
STORY

変遷のストーリー — なぜ紋が分かれたのか

7つの家紋は、一族の歴史の「分岐点」そのものだ。なぜ同じ一族なのに異なる紋を使うようになったのか。その背景には、戦乱・逃亡・降伏・復姓という、武家の運命がある。

三階菱
鎌倉時代以前

小笠原氏の三階菱から出発

清和源氏義光流の小笠原氏が本紋として使用。赤沢氏が分家する前の、いわば「家紋のルーツ」。甲斐源氏の割り菱・四つ割り菱とも同根で、武田氏とは家紋でも「親戚」の関係にある。

松皮菱に十文字
鎌倉時代 — 赤沢氏の成立

本流が松皮菱に十文字を定める

小笠原長清の孫・赤沢清経が伊豆国田方郡赤沢郷を領して赤沢氏を称した際に、小笠原氏の三階菱を変形して松皮菱とし、十文字を加えた。これが赤沢氏の正紋となり、信濃→阿波→備中の本流が代々使用した。

五七桐
室町時代 — 将軍からの賜紋

足利将軍から副紋を賜る

五七桐は、天皇→将軍→臣下へと下賜される紋の典型。赤沢氏が室町幕府の有力武将として活躍した時期に賜った可能性が高く、主紋の松皮菱と併用して副紋として使用された。

鷹の羽
戦国時代 — 讃岐への逃亡

讃岐に逃れた系統が新たな紋を採用

戦乱に追われ讃岐（香川県）に逃れた赤沢氏は、与田川口に落ち延びた。その後、丹生・三本松・白鳥地方に広がったこの系統は、本流の松皮菱ではなく「鷹の羽」と「立沢瀉」を家紋とした。本流との決別か、あるいは身を隠すための選択だったか。

立沢瀉
戦国時代 — 讃岐系の副紋

「勝ち草」を紋に選んだ讃岐の赤沢氏

鷹の羽と並んで讃岐系赤沢氏が使用。オモダカ（沢瀉）は水辺に育つ植物で、葉の形が矢じりに似ることから「勝ち草」と呼ばれ、武家に広く好まれた。逃亡先での再起を願う紋だったのかもしれない。

菱紋
戦国時代 — 武田氏への降伏

武田氏に降った稲倉の赤沢氏

信濃国筑摩郡稲倉の赤沢氏は、武田氏の侵攻に抵抗した後、降伏。家紋は松皮菱の複雑な紋から、単純な菱紋に変わった。武田氏の「割り菱」の系譜に組み込まれたことを示すのか、あるいは降伏による格下げの意味があるのか — 解釈は分かれる。

三階菱（復姓）
復姓 — 赤沢から小笠原へ

貞経の系統が小笠原に復姓し、本家の紋に戻る

赤沢貞経が小笠原氏に復姓した際、家紋も三階菱に戻った。分家として独自の紋を掲げてきた赤沢氏の中で、唯一「本家の紋に回帰」した系統。名と紋が一体であることを物語るエピソードだ。

SOURCE

原文に見る家紋の記録

冊子には、赤沢氏の家紋と移動について直接言及した記述がいくつか残っている。

一、板西城。赤沢信濃守、後剃髪して宗傳入道と称す。紋は松皮菱に十文字也。

板西城（阿波国）を拠点とした赤沢信濃守。出家して宗傳入道と名乗った人物の紋が、「松皮菱に十文字」であると明記されている。阿波の赤沢氏が本流の紋を使用していた直接的な証拠だ。

人有尊に挟られて讃岐に逃れ、与田川口に赴きたりという。其の丹生三本松白鳥地方などの赤沢氏は、其の末裔と伝え、家紋を「鷹の羽」「立沢瀉」とす。

「人有尊に挟られて」 — 追い詰められて讃岐に逃れた、という記述。そしてその末裔は、鷹の羽と立沢瀉を家紋としたと記録されている。本流の松皮菱から完全に離れた家紋を使うようになった経緯が、ここに記されている。

家紋の「変更」は、武家社会では重大な意味を持った。主君から賜る（五七桐）、本家から分かれる（松皮菱→独自紋）、本家に戻る（三階菱への復帰）、降伏の証とする（菱紋） — いずれも一族のアイデンティティの転換点。家紋は「デザイン」ではなく「宣言」だった。
